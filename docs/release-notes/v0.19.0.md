# v0.19.0 시간관리 기능이 대폭 강화되었습니다.

관리자 · 2026.03.13 14:00

v0.18.0에서 도입된 시간관리 기능에 프로젝트별 업무형 설정, 엑셀 Export 등 주요 기능이 추가되었습니다.

**프로젝트별 업무형 설정**을 통해 특정 프로젝트 코드(예: A25_05591)에 대한 기본 업무형을 지정할 수 있습니다. 시간관리 탭 상단의 파란색 톱니바퀴 아이콘을 클릭하면 프로젝트를 선택하고(프로젝트명 + 코드로 표시) 업무형(작업/개발)을 지정할 수 있습니다. 업무형 자동 분류는 4단계 우선순위로 동작합니다: 1) 프리셋 작업 → "작업", 2) 프로젝트별 설정, 3) 카테고리별 설정, 4) 기본 업무형.

**엑셀 Export 기능**으로 시간관리 데이터를 `시간관리_YYYY-MM-DD.xlsx` 형식의 엑셀 파일로 내보낼 수 있습니다. 초록색 "엑셀 Export" 버튼을 클릭하면 프로젝트 코드는 프로젝트명으로 변환되어 표시되며, 모든 컬럼 데이터와 함께 마지막 행에 시간 합계가 자동으로 추가됩니다. 일일 업무 보고서나 프로젝트별 시간 집계에 유용하게 활용할 수 있습니다.

카테고리별 업무형 기본 설정에 "질의응답" 카테고리가 추가되었으며, 기존 "테스트오류수정", "센터오류수정"과 함께 자동으로 "작업" 업무형으로 분류됩니다. 설정 페이지의 "모든 데이터 초기화" 안내문이 개선되어 게시글(Q&A)은 서버에 저장되어 초기화되지 않음을 명시합니다.

시간관리 페이지는 네비게이션 탭 또는 Alt+5 단축키로 접근할 수 있습니다.
