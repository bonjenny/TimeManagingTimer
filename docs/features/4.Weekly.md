# 주간 일정 (리포트)

주간 일정 페이지는 기록된 업무 내역을 기간별로 조회하고, 보고서 작성을 위한 통계 데이터를 제공합니다.

## 1. 기간 필터

상단 필터 버튼을 통해 조회할 기간을 선택할 수 있습니다.

| 옵션 | 설명 |
|---|---|
| **오늘** | 당일(00:00~24:00) 기록만 조회 |
| **이번 주** | 이번 주 월요일 ~ 일요일 기록 조회 |
| **이번 달** | 이번 달 1일 ~ 말일 기록 조회 |
| **직접 지정** | 시작일과 종료일을 사용자가 직접 선택 |

## 2. 데이터 집계 및 표시

선택한 기간 내의 모든 작업 기록(LocalStorage 데이터)을 분석하여 다음과 같이 표시합니다.

### 요약 카드
- **총 업무 시간**: 선택된 기간 동안의 총 작업 시간을 '분' 단위와 'HH:MM' 형식으로 표시합니다.

### 상세 테이블
작업을 **게시판 번호** 기준으로 그룹화하여 보여줍니다.

| 컬럼 | 설명 |
|---|---|
| **게시판 번호** | 작업의 그룹 키 (없으면 '미지정') |
| **업무 개수** | 해당 그룹에 속한 작업 세션 수 |
| **총 소요 시간** | 해당 그룹의 누적 작업 시간 |
| **포함된 업무** | 그룹에 포함된 작업명(Title) 목록 요약 |

## 3. 데이터 내보내기

- **CSV 다운로드**: 조회된 통계 데이터를 CSV 파일(`time_report_YYYY-MM-DD.csv`)로 다운로드할 수 있습니다. 엑셀에서 열어 주간 보고서 작성 시 활용할 수 있습니다.
