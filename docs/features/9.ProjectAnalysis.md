# 프로젝트 분석 (v0.17.0~)

프로젝트 분석 페이지는 기록된 업무를 **프로젝트 → 작업** 단위로 시간을 분해하고, 인터럽트 패턴을 분석하는 대시보드입니다.

## 1. 접근 방법

- **탭**: 상단 메뉴에서 "프로젝트 분석" 클릭
- **단축키**: `Alt + 4`

## 2. 시간 분해 (Time Breakdown)

### 기간 선택
- **월간**: 해당 월 전체 집계. 이전/다음 달 네비게이션 지원.
- **분기**: 3개월 단위 집계. 이전/다음 분기 네비게이션 지원.
- **직접 지정**: 시작일~종료일 수동 선택.

### 프로젝트 필터
- Autocomplete 검색으로 특정 프로젝트만 조회 가능.
- 비우면 전체 프로젝트 표시.

### 카테고리별 비율
- 전체 업무의 카테고리 비율을 스택 바로 시각화.
- 호버 시 카테고리명, 시간, 비율 표시.

### 프로젝트별 작업 테이블
- 프로젝트 내 작업명(title)별 소요 시간, 세션 수, 비율(%) 표시.
- 수평 바 차트로 비율 시각화.
- 작업은 소요 시간 내림차순 정렬.

## 3. 세션 드릴다운

작업 행 클릭 시 해당 작업의 개별 세션 목록이 Collapse로 펼쳐집니다.

### 세션 정보
| 열 | 설명 |
|---|---|
| 날짜 | 세션 시작 날짜 (MM/DD) |
| 시간 | 시작~종료 시간 |
| 소요 | 점심시간/일시정지 제외 소요 시간 |
| 비고 | note 필드 — 인라인 편집 가능 |

### 메모 인라인 편집
- 비고 열 클릭 시 TextField로 전환.
- Enter / ✓로 저장, Escape / ✕로 취소.
- 영역 바깥 클릭 시 자동 저장 (ClickAwayListener).

## 4. 인터럽트 분석

### 키워드 필터
- 검색창에 키워드를 입력하여 인터럽트로 추적할 작업을 지정.
- 작업명, 카테고리, 프로젝트 코드에서 부분 일치 검색.
- 복수 키워드 추가 가능 (OR 조건).
- 자동완성: 해당 기간의 작업명 + 카테고리 목록 제안.

### 요약 통계
- 총 인터럽트 건수
- 총 소요시간
- 최다 발생 주
- 피크 시간대 (2시간 윈도우)

### 주차별 히트맵
- 월~금 × 주차 매트릭스.
- 색상 강도: 소요 시간 비례 (빨강 계열).
- 각 셀에 분 단위 시간 표시.
- 호버 시 상세 시간 표시.

### 시간대별 분포
- 06~24시 구간별 막대 차트.
- 색상 구분: >70% 빨강, >40% 주황, 기본 primary.
- 호버 시 정확한 시간 표시.

### 발생 이력 테이블
- 매칭된 모든 세션을 시간순 나열.
- 날짜, 시간, 작업명, 카테고리, 소요, 비고 표시.

## 5. 타이머 완료 시 메모 입력

일간 타이머의 ActiveTimer에서 "완료" 클릭 시:
1. 메모 입력 필드가 Collapse로 표시됨.
2. 메모 입력 후 "저장" 또는 Enter → note 저장 후 완료.
3. "건너뛰기" 또는 Escape → 메모 없이 즉시 완료.

## 6. 내보내기

### 클립보드 복사
- 프로젝트별 작업 분해 결과를 텍스트 형식으로 복사.

### CSV 다운로드
- 프로젝트코드, 프로젝트명, 작업명, 카테고리, 세션수, 소요시간(분), 비율(%) 포함.
- BOM 포함으로 Excel 한글 호환.

## 7. 데이터

- 기존 `useTimerStore`의 `logs` 데이터를 사용 (별도 저장소 없음).
- `useProjectStore`에서 프로젝트명 조회.
- `useCategoryStore`에서 인터럽트 키워드 자동완성용 카테고리 목록 사용.
- note 수정은 `useTimerStore.updateLog()`를 통해 즉시 반영.
