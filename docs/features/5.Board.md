# 건의사항 게시판

> **현재 준비 중인 기능입니다.**

건의사항 게시판은 앱 개선 아이디어나 버그 리포트를 작성하는 공간입니다.
서버 없이 로컬 환경에서 동작하므로, 실제 개발자에게 전송되는 기능은 추후 구현될 예정입니다.

## 예정 기능

- **익명 글쓰기**: 별도 로그인 없이 닉네임으로 작성
- **비밀번호 관리**: 게시글 수정/삭제를 위한 비밀번호 설정
- **댓글**: 게시글에 대한 댓글 작성
