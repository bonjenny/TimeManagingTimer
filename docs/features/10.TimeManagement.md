# 시간관리 (Time Management)

## 개요

시간관리 탭은 일간 타이머의 작업 기록을 체계적으로 정리하고 관리할 수 있는 표 형식의 인터페이스를 제공합니다. 일일 업무 보고나 시간 집계가 필요한 경우 유용하게 활용할 수 있습니다.

## 주요 기능

### 1. 데이터 자동 불러오기

- 시간관리 탭 접근 시 해당 날짜의 일간 타이머 데이터를 자동으로 변환하여 로드
- 같은 작업명의 로그는 자동으로 그룹화되어 시간이 합산됨
- 날짜 변경 시에도 자동으로 해당 날짜의 데이터 로드

### 2. 표 형식 데이터 관리

**컬럼 구성**:
- **작업**: 프로젝트명 (프로젝트 코드로 저장, 이름으로 표시)
- **업무형**: 작업 또는 개발 (드롭다운 선택)
- **거래형(일정명)**: 작업 제목
- **카테고리 코드**: 숫자 코드 (예: 01, 05, 13)
- **카테고리명**: 카테고리 이름
- **시간(분)**: 작업 소요 시간
- **비고**: 메모 및 특이사항

**인라인 편집**:
- 셀을 클릭하여 직접 수정 가능
- Enter 또는 포커스 아웃 시 자동 저장
- 프로젝트, 카테고리, 업무형은 드롭다운으로 선택

### 3. 업무형 자동 분류

작업의 업무형은 다음 우선순위로 자동 결정됩니다:

1. **프리셋 작업** (최우선): 항상 "작업"
2. **프로젝트별 설정**: 특정 프로젝트 코드에 대한 업무형
3. **카테고리별 설정**: 특정 카테고리에 대한 업무형
4. **기본 업무형**: 위 조건에 해당하지 않는 경우

**기본 설정**:
- 테스트오류수정 → "작업"
- 센터오류수정 → "작업"
- 질의응답 → "작업"
- A25_05591 프로젝트 → "작업"

### 4. 설정 관리

#### 기본 업무형 설정
- 시간관리 탭 상단에서 드롭다운으로 선택
- 새로 추가되는 행과 자동 변환되는 데이터에 적용

#### 카테고리별 업무형 설정
- 특정 카테고리에 대한 기본 업무형 지정
- 설정 다이얼로그에서 추가/삭제 관리
- 예: "개발" 카테고리는 항상 "개발" 업무형

#### 프로젝트별 업무형 설정
- 특정 프로젝트 코드에 대한 기본 업무형 지정
- 설정 다이얼로그에서 추가/삭제 관리
- 예: A25_05591 프로젝트는 항상 "작업" 업무형

### 5. 행 추가/삭제

- **새 행 추가**: 빈 행을 추가하여 수동으로 데이터 입력
- **체크박스 선택 삭제**: 여러 행을 선택하여 일괄 삭제
- **전체 선택**: 헤더의 체크박스로 전체 행 선택/해제

### 6. 엑셀 Export

- **파일명**: `시간관리_YYYY-MM-DD.xlsx`
- **내용**: 
  - 프로젝트 코드 → 프로젝트명으로 변환
  - 모든 컬럼 데이터 포함
  - 마지막 행에 시간 합계 자동 추가
- **버튼**: 초록색 "엑셀 Export" 버튼
- 데이터가 없을 때는 버튼 비활성화

### 7. 날짜 네비게이션

- **이전/다음 날**: 화살표 버튼으로 날짜 이동
- **오늘 버튼**: 현재 날짜로 빠르게 이동
- **날짜 선택**: 달력 아이콘 클릭하여 특정 날짜 선택
- 날짜 변경 시 해당 날짜의 데이터 자동 로드

## 데이터 독립성

- 시간관리 탭의 데이터는 일간 타이머와 **별도로 관리**됨
- 시간관리 탭에서 수정해도 원본 일간 타이머 데이터는 변경되지 않음
- 일간 타이머 데이터를 다시 불러오면 새로운 로그만 추가됨 (중복 방지)

## 카테고리 코드 매핑

시스템에서 사용하는 카테고리 코드는 다음과 같습니다:

- 01: 기획
- 02: 기획테스트
- 03: 기획검수
- 05: 회의
- 13: 개발
- 14: 개발테스트
- 9999: 기타 (기본값)

전체 목록은 46개 카테고리를 포함하며, 상세 내용은 `categoryCodeMap.ts`를 참조하세요.

## 사용 시나리오

### 시나리오 1: 일일 업무 보고
1. 시간관리 탭 접근 (자동으로 오늘 데이터 로드)
2. 필요시 프로젝트명, 업무형, 카테고리 수정
3. 비고란에 특이사항 입력
4. "엑셀 Export" 버튼으로 보고서 생성

### 시나리오 2: 프로젝트별 시간 집계
1. 특정 날짜 선택
2. 프로젝트별로 시간 확인
3. 필요시 수동으로 행 추가하여 누락된 작업 입력
4. 엑셀로 내보내어 프로젝트 리포트 작성

### 시나리오 3: 업무형 분류 자동화
1. 설정에서 자주 사용하는 카테고리/프로젝트의 업무형 지정
2. 이후 자동 불러오기 시 업무형이 자동으로 분류됨
3. 수동 수정 작업 최소화

## 주의사항

- 일간 타이머에서 불러온 데이터는 원본 로그 ID로 연결됩니다
- 동일한 로그를 중복으로 불러오지 않습니다
- 수동으로 추가한 행은 원본 로그 ID가 없습니다
- 엑셀 Export 시 체크박스 정보는 포함되지 않습니다
