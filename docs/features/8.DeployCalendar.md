# 배포 캘린더 (v0.14.0~)

배포 캘린더 탭에서는 배포 일정을 월간 그리드로 관리하고, HTML 테이블로 복사해 웹에디터에 붙여넣을 수 있습니다.

## 1. 접근 방법

- **탭**: 상단 메뉴에서 "배포 캘린더" 클릭
- **단축키**: `Alt + 3`

## 2. 화면 구성

- **기본 표시**: 오늘 기준으로 선택한 **주 수**만큼 표시 (월~금). 기본값 2주(저번주~이번주).
- **표시 주 수 설정**: **설정 > 배포 캘린더**에서 1주 / 2주 / 3주 / 4주 중 선택 가능. 자동 저장.
- **주차 표시**: 예) "2026년 2월 1주차 ~ 2주차"
- **네비게이션**: 이전 주 / 다음 주 / 이번 주로 이동

## 3. 이벤트 관리

- **추가**: 빈 셀 클릭 → 날짜, 프로젝트 코드(자동완성), 표시명, 진행상태, 휴일 여부 입력
- **수정/삭제**: 이벤트 셀 클릭 → 모달에서 수정 또는 삭제. 모달에서 **Enter** 키로 저장 가능. 모달에서 잡 색상(기본·테마·직접 선택) 편집 가능.
- **우클릭 삭제**: 이벤트 칩에서 우클릭 → 컨텍스트 메뉴에서 "삭제" 선택 시 해당 이벤트만 삭제
- **드래그앤드롭**: 이벤트 칩을 다른 날짜 셀로 드래그하여 일자 이동
- **휴일**: 휴일 체크 시 해당 셀에 빨간색 텍스트로 표시 (예: 신정, 연차)

## 4. 잡 색상 설정

- **잡 색상 설정** 버튼으로 다이얼로그 열기
- 잡(프로젝트 코드)별로 배경색 지정
- **기본 색상**: 8색 파스텔 구역 + **테마 색상**: 설정 > 컬러 팔레트 테마 (다크모드 시 자동 보정)
- 컬러피커로 직접 선택 가능

## 5. HTML 복사 (v0.14.2~)

- **HTML 복사** 버튼 클릭 시 현재 표시된 주 수만큼 캘린더를 HTML 테이블로 클립보드에 복사
- ecount 웹에디터, Notion, Excel 등에 붙여넣기 시 아래 스타일이 유지됨

### 내보내기 스타일

- **전체**: 얇은 회색 테두리, 둥근 모서리(8px), 고정 너비(835px), colgroup으로 열 너비 유지
- **날짜 헤더**: 옅은 회색 배경(#f3f4f6), 가운데 정렬, 고정 높이(36px), 상하 패딩
- **바디 셀**: 흰 배경, 최소 높이 72px, 칸·라벨 좌우 패딩
- **작업 라벨**: 잡별 배경색(td bgcolor + 인라인 스타일), 둥근 모서리(6px), 패딩(8px 12px 10px 12px), 라벨 간 margin(6px), 긴 텍스트 말줄임표
- **휴일**: 빨간색(#ca3626) 텍스트, 가운데 정렬

## 6. 데이터

- 모든 데이터는 **LocalStorage**에 저장 (`timekeeper-deploy-calendar`)
- 프로젝트 코드는 기존 **설정 > 프로젝트** 및 일간 타이머에서 사용한 코드가 자동완성에 표시됨
