# 단축키 목록

키보드 단축키를 사용하여 마우스 없이도 앱을 빠르게 조작할 수 있습니다.

## 현재 지원되는 단축키

| 단축키 | 기능 | 설명 |
|---|---|---|
| `Enter` | **타이머 시작** | 타이머 입력창에 포커스가 있을 때, 입력을 완료하고 타이머를 시작합니다. |
| `F8` | **새 작업 추가** | 어디서든 새 작업 추가 팝업을 엽니다. |
| `Alt + N` | **입력창 포커스** | 일간 타이머 입력창 포커스. 다른 페이지면 일간으로 이동 후 포커스. |
| `Alt + S` | **일시정지/재개** | 현재 진행 중인 타이머 일시정지 또는 재개. |
| `Alt + 1` | **일간 타이머** | 일간 타이머 페이지로 이동. |
| `Alt + 2` | **주간 일정** | 주간 일정 페이지로 이동. |
| `Alt + 3` | **배포 캘린더** | 배포 캘린더 페이지로 이동. (v0.14.0~) |
| `Alt + T` | **오늘로 이동** | 일간 페이지에서 선택 날짜를 오늘로 변경. |
