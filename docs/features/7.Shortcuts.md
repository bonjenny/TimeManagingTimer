# 단축키 목록

키보드 단축키를 사용하여 마우스 없이도 앱을 빠르게 조작할 수 있습니다.

## 현재 지원되는 단축키

| 단축키 | 기능 | 설명 |
|---|---|---|
| `Enter` | **타이머 시작** | 타이머 입력창에 포커스가 있을 때, 입력을 완료하고 타이머를 시작합니다. |
| `F8` | **새 작업 추가** | (TODO) 어디서든 새 작업 추가 팝업을 엽니다. |

아래 단축키들은 향후 업데이트에서 지원될 예정입니다.

- **Alt + N**: 새 작업 추가 (입력창 포커스)
- **Alt + S**: 현재 진행 중인 타이머 일시정지/재개
- **Alt + T**: 오늘 날짜로 이동
- **Alt + 1 / 2**: 페이지 이동 (일간/주간)
