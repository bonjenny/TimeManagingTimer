# 시작하기

업무 시간 관리는 회사에서 수행하는 작업을 실시간으로 기록하고 관리하는 웹 애플리케이션입니다. 타이머 기반으로 작업 시간을 정확하게 측정하고, 일간/주간 단위로 작업 내역을 체계적으로 관리할 수 있습니다.

**모든 데이터는 사용자의 브라우저(LocalStorage)에만 저장됩니다.** 별도의 서버나 DB를 사용하지 않으므로 보안 걱정 없이 개인적으로 안전하게 사용할 수 있습니다.

## 핵심 기능 개요

| 기능 | 설명 | 관련 문서 |
|---|---|---|
| **타이머 기반 기록** | 작업 시작/정지로 정확한 시간 측정 | [일간 기록](2.DailyRecord.md) |
| **작업 프리셋** | 최근 작업 기반 자동 프리셋 생성 | [작업 프리셋](3.WorkPreset.md) |
| **간트차트 시각화** | 하루 작업을 시간대별로 시각화 (드래그 지원) | [일간 기록](2.DailyRecord.md) |
| **주간 리포트** | 주간 작업 내역을 기간별로 조회 및 내보내기 | [주간 일정](4.Weekly.md) |
| **데이터 관리** | JSON 파일로 백업 및 복원 (필수) | [설정](6.Settings.md) |

### 데이터 저장 및 주의사항

| 항목 | 내용 |
|---|---|
| **저장 방식** | 브라우저 LocalStorage (서버 저장 X) |
| **장점** | 로그인 불필요, 인터넷 없이 사용 가능, 개인 정보 보호 |
| **주의** | 브라우저 캐시 삭제 시 데이터가 사라질 수 있음 |
| **백업** | **설정 > 데이터 내보내기**를 통해 주기적으로 JSON 파일을 저장하세요. |

## 화면 구성

### 전체 레이아웃 (3단 구성)

앱은 데스크톱 환경에서 효율적인 작업을 위해 3단 구성 레이아웃을 제공합니다.

```
┌─────────────────────────────────────────────────────┐
│                    네비게이션 바                      │
├───────────┬─────────────────────────────────────────┤
│           │  [상단] 간트 차트 (타임라인 시각화)       │
│  [좌측]    ├─────────────────────────────────────────┤
│  작업     │  [중단] 타이머 입력 + 활성 타이머         │
│  프리셋   ├─────────────────────────────────────────┤
│  패널     │  [하단] 작업 기록 리스트                     │
└───────────┴─────────────────────────────────────────┘
```

| 영역 | 주요 기능 |
|---|---|
| **네비게이션 바** | 페이지 이동 (일간/주간/건의사항/설정) |
| **작업 프리셋** | 최근 사용한 작업 목록 표시, 클릭 시 바로 시작 |
| **중앙 (상단)** | 하루 일정을 시각화한 간트 차트 (드래그로 작업 추가) |
| **중앙 (중단)** | 새 작업 입력 및 현재 진행 중인 타이머 표시/제어 |
| **중앙 (하단)** | 상세 작업 기록 리스트 (수정, 삭제, 재시작) |

### 네비게이션 메뉴

| 메뉴 | 기능 |
|---|---|
| **일간 타이머** | 메인 작업 화면 (타이머, 기록, 간트차트) |
| **주간 일정** | 기간별 작업 통계 및 리포트 |
| **건의사항** | (준비 중) 피드백 게시판 |
| **설정** | 테마, 데이터 백업/복원, 환경 설정 |

## 모바일 사용

앱은 반응형으로 설계되어 모바일 기기에서도 사용할 수 있습니다. 화면 너비가 좁아지면 3단 구성이 **세로 배치**로 자동 전환됩니다.

1. **프리셋 패널**: 상단으로 이동
2. **타이머/간트**: 중앙 배치
3. **리스트**: 하단 배치
