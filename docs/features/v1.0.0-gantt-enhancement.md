# v1.0.0 간트 차트 사용성 개선 기능 명세

## 1. 개요
간트 차트에서 드래그 앤 드롭으로 작업을 생성하거나 수정할 때, 사용자의 마우스 조작 실수로 인한 미세한 시간 공백이나 의도치 않은 중복 입력을 방지하기 위한 보조 기능을 추가합니다.

## 2. 상세 기능

### 2.1. 시간 자동 조정 (Smart Snap)
사용자가 시간을 선택할 때, 기존 작업들의 시간 정보를 기준으로 근접한 시간에 자동으로 맞추어 주는 기능입니다.

- **트리거**: 
  - 새 작업 드래그 생성 중 (`MouseMove`)
  - 기존 작업 리사이징 중 (`MouseMove`)
- **조건**: 
  - 드래그 중인 끝점(시작 시간 또는 종료 시간)이 기존 타 작업의 `시작 시간` 또는 `종료 시간`과 `±15분` 이내 차이가 날 경우.
- **동작**: 
  - 드래그 중인 시간을 기준이 되는 기존 작업의 시간과 동일하게 강제 조정합니다.
  - 이를 통해 작업 간의 빈틈 없는 연결(이어 붙이기)을 쉽게 할 수 있습니다.

### 2.2. 중복 입력 경고 (Overlap Warning)
작업 시간이 겹칠 경우 사용자에게 이를 인지시키고 의사결정을 묻습니다.

- **트리거**: 
  - 드래그 종료 후 저장 시점 (`MouseUp`)
  - 수동 시간 입력 저장 시점
- **조건**: 
  - 새로 설정된 시간 범위가 기존의 다른 작업 시간 범위와 교차(Overlap)하는 경우.
  - 단, 시작 시간과 종료 시간이 맞닿아 있는 경우(예: A종료 13:00, B시작 13:00)는 중복으로 간주하지 않음.
- **동작**: 
  - "입력하신 시간이 기존 작업과 겹칩니다. 저장하시겠습니까?" 메시지의 확인 창(Confirm/Alert)을 표시합니다.
  - [확인]: 겹침을 허용하고 저장합니다.
  - [취소]: 저장을 취소하고 이전 상태로 되돌리거나 수정 모드를 유지합니다.

## 3. 예외 처리
- **스냅 우선순위**: 여러 시간 점이 스냅 범위 내에 있을 경우, 가장 가까운 시간을 우선합니다.
- **자신 제외**: 중복 검사 및 스냅 기준에서 수정 중인 자기 자신은 제외합니다.
