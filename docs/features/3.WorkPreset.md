# 작업 프리셋

작업 프리셋은 사용자의 최근 작업 패턴을 분석하여 자동으로 생성되는 '바로가기' 목록입니다. 화면 좌측(모바일은 상단) 패널에서 확인할 수 있습니다.

## 1. 프리셋 자동 생성

별도의 등록 과정 없이, 작업을 기록하면 자동으로 프리셋이 생성됩니다.

- **생성 기준**: 최근 30일 이내에 기록된 작업 중
- **정렬**: 최신 작업 순으로 정렬
- **개수 제한**: 최대 10개까지 표시
- **중복 제거**: 동일한 작업명(Title)은 하나만 표시

## 2. 프리셋 활용

프리셋 목록의 항목을 클릭하면 즉시 해당 설정으로 타이머가 시작됩니다.

- **빠른 시작**: 작업명, 게시판 번호, 카테고리가 자동으로 입력된 상태로 새 타이머가 시작됩니다.
- **편의성**: 매번 동일한 업무 내용을 입력할 필요가 없습니다.

## 3. 즐겨찾기 (Favorites)

중요한 작업은 별(☆) 아이콘을 눌러 즐겨찾기 할 수 있습니다.

- **기능**: 즐겨찾기 된 항목은 목록 상단에 고정되거나 별도로 강조 표시됩니다.
- **상태 저장**: 즐겨찾기 상태는 브라우저 LocalStorage에 저장되어 유지됩니다.
