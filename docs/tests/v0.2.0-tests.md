## 1. F8 단축키 새 작업 추가 팝업

### TC-201: F8 단축키로 모달 열기
- **전제조건**: 앱이 정상적으로 로드됨
- **테스트 단계**:
  1. 일간 타이머 페이지에서 F8 키 누름
  2. 모달 표시 확인
- **예상 결과**: 
  - "새 작업 시작" 모달이 나타남
  - 제목 입력창에 자동 포커스
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-202: 다른 페이지에서 F8
- **전제조건**: 설정 페이지에 있음
- **테스트 단계**:
  1. F8 키 누름
- **예상 결과**: 
  - 설정 페이지에서도 모달이 열림
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-203: 모달에서 작업 시작
- **전제조건**: F8 모달이 열려 있음
- **테스트 단계**:
  1. 작업 제목 "F8 테스트 작업" 입력
  2. Enter 키 누름
- **예상 결과**: 
  - 모달이 닫힘
  - 새 타이머가 시작됨
  - 활성 타이머에 "F8 테스트 작업" 표시
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-204: 자동 완성 동작
- **전제조건**: 이전에 기록한 작업이 있음
- **테스트 단계**:
  1. F8 모달 열기
  2. 이전 작업명의 첫 글자 입력
- **예상 결과**: 
  - 자동 완성 드롭다운에 해당 작업이 표시됨
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-205: ESC로 모달 닫기
- **전제조건**: F8 모달이 열려 있음
- **테스트 단계**:
  1. ESC 키 누름
- **예상 결과**: 
  - 모달이 닫힘
  - 타이머가 시작되지 않음
- **실제 결과**: [ ] Pass / [ ] Fail

## 2. 간트 차트 막대 Resize

### TC-206: 완료된 작업 리사이즈 핸들 표시
- **전제조건**: 완료된 작업이 간트 차트에 있음
- **테스트 단계**:
  1. 완료된 작업 막대에 마우스 호버
  2. 막대 좌측 끝에 커서 이동
  3. 막대 우측 끝에 커서 이동
- **예상 결과**: 
  - 좌우 끝에서 커서가 `↔` (ew-resize)로 변경
  - 호버 시 핸들 영역이 약간 밝아짐
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-207: 종료 시간 늘리기
- **전제조건**: 완료된 작업 (09:00~10:00)이 있음
- **테스트 단계**:
  1. 막대 우측 끝을 드래그
  2. 오른쪽으로 30분 정도 이동
  3. 마우스 놓기
- **예상 결과**: 
  - 드래그 중 막대가 실시간으로 늘어남
  - 종료 시간이 10:30으로 변경됨
  - LocalStorage에 저장됨
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-208: 시작 시간 당기기
- **전제조건**: 완료된 작업 (09:00~10:00)이 있음
- **테스트 단계**:
  1. 막대 좌측 끝을 드래그
  2. 왼쪽으로 30분 정도 이동
  3. 마우스 놓기
- **예상 결과**: 
  - 시작 시간이 08:30으로 변경됨
  - 작업 시간이 1시간 30분으로 증가
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-209: 최소 시간 제한 (5분)
- **전제조건**: 완료된 작업이 있음
- **테스트 단계**:
  1. 막대 우측 끝을 왼쪽으로 최대한 드래그
- **예상 결과**: 
  - 막대가 5분 이하로 줄어들지 않음
  - 시작 시간과 종료 시간 사이 최소 5분 유지
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-210: 진행 중인 작업 리사이즈 불가
- **전제조건**: 현재 진행 중인 작업이 있음
- **테스트 단계**:
  1. 진행 중인 작업 막대에 마우스 호버
- **예상 결과**: 
  - 리사이즈 핸들이 표시되지 않음
  - 커서가 ew-resize로 변경되지 않음
- **실제 결과**: [ ] Pass / [ ] Fail

## 3. 날짜 이동 기능

### TC-211: 이전 날짜 이동
- **전제조건**: 일간 타임라인이 오늘 날짜를 표시 중
- **테스트 단계**:
  1. '<' 버튼 클릭
- **예상 결과**: 
  - 날짜가 어제로 변경됨
  - 해당 날짜의 기록이 표시됨
  - 현재 시간 표시선이 사라짐
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-212: 다음 날짜 이동
- **전제조건**: 어제 날짜를 보고 있음
- **테스트 단계**:
  1. '>' 버튼 클릭
- **예상 결과**: 
  - 날짜가 오늘로 변경됨
  - 현재 시간 표시선이 나타남
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-213: 오늘로 이동 버튼
- **전제조건**: 일주일 전 날짜를 보고 있음
- **테스트 단계**:
  1. '오늘' 버튼 (달력 아이콘) 클릭
- **예상 결과**: 
  - 오늘 날짜로 즉시 이동
  - '오늘' 버튼이 사라짐 (오늘이므로)
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-214: 오늘 날짜 강조 표시
- **전제조건**: 오늘 날짜를 보고 있음
- **테스트 단계**:
  1. 날짜 표시 영역 확인
- **예상 결과**: 
  - 검정 배경에 흰색 텍스트로 표시
  - '오늘' 버튼이 표시되지 않음
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-215: 과거 날짜 기록 조회
- **전제조건**: 3일 전에 기록한 작업이 있음
- **테스트 단계**:
  1. '<' 버튼을 3번 클릭하여 3일 전으로 이동
- **예상 결과**: 
  - 3일 전의 작업 기록이 간트 차트에 표시됨
  - 활성 타이머는 표시되지 않음
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-216: 과거 날짜 기록 리사이즈
- **전제조건**: 과거 날짜에 완료된 작업이 있음
- **테스트 단계**:
  1. 해당 날짜로 이동
  2. 작업 막대 리사이즈 수행
- **예상 결과**: 
  - 과거 기록도 리사이즈 가능
  - 변경 사항이 저장됨
- **실제 결과**: [ ] Pass / [ ] Fail
