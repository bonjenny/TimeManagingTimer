## 2. 일간 타이머 테스트

### TC-003: 새 작업 시작
- **전제조건**: 일간 타이머 페이지, 활성 타이머 없음
- **테스트 단계**:
  1. 입력창에 "테스트 작업" 입력
  2. Enter 키 누름
- **예상 결과**: 
  - 활성 타이머 영역에 "테스트 작업" 표시
  - 타이머가 00:00부터 시작
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-004: 타이머 일시정지 및 재개
- **전제조건**: 활성 타이머가 실행 중
- **테스트 단계**:
  1. '일시정지' 버튼 클릭
  2. 10초 대기
  3. '재개' 버튼 클릭
- **예상 결과**: 
  - 일시정지 시 "일시정지" 상태 표시
  - 재개 시 타이머가 이어서 진행
  - 일시정지 시간은 총 시간에서 제외
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-005: 타이머 완료
- **전제조건**: 활성 타이머가 실행 중
- **테스트 단계**:
  1. '완료' 버튼 클릭
- **예상 결과**: 
  - 활성 타이머 영역이 비워짐
  - 작업 기록 리스트에 완료된 작업 추가
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-006: 타이머 중단 (삭제)
- **전제조건**: 활성 타이머가 실행 중
- **테스트 단계**:
  1. ⬛ (중단) 버튼 클릭
- **예상 결과**: 
  - 활성 타이머가 사라짐
  - 작업 기록에 추가되지 않음
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-006-1: 타이머 시작 시 초 표시 (00:00:00 형식)
- **전제조건**: 활성 타이머 없음
- **테스트 단계**:
  1. 입력창에 "초 테스트" 입력
  2. Enter 키 누름
  3. 타이머 시간 표시 영역 확인
- **예상 결과**: 
  - 시간이 00:00:00 형태로 표시됨
  - 처음 5초 동안 초가 1초 단위로 업데이트됨 (00:00:01, 00:00:02, ...)
  - 초 부분이 잘리지 않고 완전히 표시됨
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-006-2: 5초 이후 초 fade out
- **전제조건**: 타이머가 실행 중 (5초 이상 경과)
- **테스트 단계**:
  1. 타이머 시작 후 5초 대기
  2. 시간 표시 영역 관찰
- **예상 결과**: 
  - 5초 이후 초 부분(:SS)이 fade out 애니메이션과 함께 사라짐
  - 최종적으로 00:00 형태만 표시됨
  - fade out 후 시간 표시가 버튼과 자연스럽게 붙어 표시됨
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-006-3: 일시정지 후 재개 시 초 업데이트
- **전제조건**: 타이머가 실행 중 (5초 미만)
- **테스트 단계**:
  1. 타이머 시작 (2초 경과)
  2. 일시정지 버튼 클릭
  3. 5초 대기
  4. 재개 버튼 클릭
  5. 시간 표시 확인
- **예상 결과**: 
  - 일시정지 시간은 총 시간에서 제외
  - 재개 후에도 초가 1초 단위로 업데이트됨 (남은 3초 동안)
  - 총 5초 경과 후 초 부분이 fade out
- **실제 결과**: [ ] Pass / [ ] Fail
