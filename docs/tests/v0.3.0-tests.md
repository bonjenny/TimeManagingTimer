## 1. 즐겨찾기 기능

### TC-301: 즐겨찾기 추가
- **전제조건**: 프리셋 목록에 항목이 있음
- **테스트 단계**:
  1. 프리셋 항목의 빈 별(☆) 아이콘 클릭
- **예상 결과**: 
  - 아이콘이 노란 별(⭐)로 변경
  - 해당 항목이 목록 상단으로 이동
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-302: 즐겨찾기 해제
- **전제조건**: 즐겨찾기된 프리셋이 있음
- **테스트 단계**:
  1. 노란 별(⭐) 아이콘 클릭
- **예상 결과**: 
  - 아이콘이 빈 별(☆)로 변경
  - 해당 항목이 원래 위치로 이동
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-303: 즐겨찾기 영구 저장
- **전제조건**: 프리셋을 즐겨찾기함
- **테스트 단계**:
  1. 프리셋 즐겨찾기
  2. 브라우저 새로고침 (F5)
  3. 프리셋 목록 확인
- **예상 결과**: 
  - 새로고침 후에도 즐겨찾기 상태 유지
  - 즐겨찾기 항목이 상단에 표시
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-304: 여러 개 즐겨찾기
- **전제조건**: 프리셋이 5개 이상 있음
- **테스트 단계**:
  1. 3개의 프리셋을 순서대로 즐겨찾기
- **예상 결과**: 
  - 3개 모두 목록 상단에 표시
  - 즐겨찾기 순서 유지
- **실제 결과**: [ ] Pass / [ ] Fail

## 2. 프리셋 수동 추가

### TC-305: 수동 프리셋 추가 모달 열기
- **전제조건**: 프리셋 패널이 표시됨
- **테스트 단계**:
  1. 헤더의 + 버튼 클릭
- **예상 결과**: 
  - "새 프리셋 추가" 모달 표시
  - 작업 제목 입력창에 포커스
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-306: 필수 항목만 입력하여 추가
- **전제조건**: 추가 모달이 열려 있음
- **테스트 단계**:
  1. 작업 제목에 "테스트 프리셋" 입력
  2. 추가 버튼 클릭
- **예상 결과**: 
  - 모달 닫힘
  - 프리셋 목록에 "테스트 프리셋" 추가
  - "수동" 라벨 표시
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-307: 모든 항목 입력하여 추가
- **전제조건**: 추가 모달이 열려 있음
- **테스트 단계**:
  1. 작업 제목: "주간 회의"
  2. 게시판 번호: "12345"
  3. 카테고리: "회의"
  4. 색상: 파랑 선택
  5. 추가 버튼 클릭
- **예상 결과**: 
  - 모든 정보가 프리셋에 저장됨
  - 파란색 인디케이터 표시
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-308: 제목 없이 추가 시도
- **전제조건**: 추가 모달이 열려 있음
- **테스트 단계**:
  1. 작업 제목을 비워둠
  2. 추가 버튼 확인
- **예상 결과**: 
  - 추가 버튼이 비활성화됨
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-309: 수동 프리셋 영구 저장
- **전제조건**: 수동 프리셋을 추가함
- **테스트 단계**:
  1. 수동 프리셋 추가
  2. 브라우저 새로고침
  3. 프리셋 목록 확인
- **예상 결과**: 
  - 새로고침 후에도 수동 프리셋 유지
- **실제 결과**: [ ] Pass / [ ] Fail

  - 프리셋 좌측에 지정한 색상의 세로 바 표시
- **실제 결과**: [ ] Pass / [ ] Fail

## 4. 프리셋 수정/삭제

### TC-313: 수동 프리셋 수정 모달 열기
- **전제조건**: 수동 프리셋이 있음
- **테스트 단계**:
  1. 수동 프리셋의 연필(✏️) 아이콘 클릭
- **예상 결과**: 
  - "프리셋 수정" 모달 표시
  - 기존 정보가 입력되어 있음
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-314: 프리셋 정보 수정
- **전제조건**: 수정 모달이 열려 있음
- **테스트 단계**:
  1. 작업 제목을 "수정된 프리셋"으로 변경
  2. 수정 버튼 클릭
- **예상 결과**: 
  - 프리셋 목록에서 제목이 변경됨
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-315: 프리셋 삭제
- **전제조건**: 수동 프리셋의 수정 모달이 열림
- **테스트 단계**:
  1. 좌측 하단의 "삭제" 버튼 클릭
- **예상 결과**: 
  - 모달 닫힘
  - 프리셋 목록에서 해당 항목 제거
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-316: 자동 프리셋 수정 불가
- **전제조건**: 자동 생성된 프리셋이 있음
- **테스트 단계**:
  1. 자동 프리셋 항목 확인
- **예상 결과**: 
  - 연필(✏️) 아이콘이 표시되지 않음
