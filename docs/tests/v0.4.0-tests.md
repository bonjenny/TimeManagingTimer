## 1. 주간 단위 이동

### TC-401: 이전 주 이동
- **전제조건**: 주간 일정 페이지에서 이번 주를 보고 있음
- **테스트 단계**:
  1. '<' 버튼 클릭
- **예상 결과**: 
  - 날짜가 1주 전으로 변경됨
  - 해당 주의 데이터 표시
  - "이번 주" 버튼 표시
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-402: 다음 주 이동
- **전제조건**: 지난 주를 보고 있음
- **테스트 단계**:
  1. '>' 버튼 클릭
- **예상 결과**: 
  - 날짜가 1주 후로 변경됨
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-403: 이번 주로 복귀
- **전제조건**: 2주 전을 보고 있음
- **테스트 단계**:
  1. "이번 주" 버튼 (달력 아이콘) 클릭
- **예상 결과**: 
  - 이번 주로 즉시 이동
  - "이번 주" 버튼 사라짐
  - 주 표시가 검정 배경으로 변경
- **실제 결과**: [ ] Pass / [ ] Fail

### TC-404: 주 범위 표시
- **전제조건**: 주간 일정 페이지 열기
- **테스트 단계**:
  1. 상단의 주 범위 확인
- **예상 결과**: 
  - "YYYY년 M/D ~ M/D" 형식으로 표시
  - 월요일 ~ 일요일 범위
- **실제 결과**: [ ] Pass / [ ] Fail
