# TimeKeeper v0.23.0 사용설명서

## 1. 일일 자동 백업 기능

만일의 사태(브라우저 캐시 삭제 등)로 인한 데이터 유실을 방지하기 위해 **일일 자동 백업 기능**이 추가되었습니다.

- **기본 동작**: 하루에 처음 앱을 열 때, 약 3초 후 전체 데이터를 담은 `timekeeper-backup-auto-YYYY-MM-DD.json` 파일이 자동으로 다운로드됩니다.
- **설정 변경**: 우측 상단의 톱니바퀴 아이콘을 눌러 `설정` 화면에 들어가면, **[데이터 관리]** 섹션에서 "일일 자동 백업 사용" 토글을 통해 기능을 켜거나 끌 수 있습니다.
- 수동으로 백업(데이터 내보내기)을 실행한 날에는 중복해서 자동 백업을 수행하지 않습니다.

## 2. 시간관리 비고 묶기

시간관리 탭에서 `일간 타이머에서 불러오기`를 실행할 때, 같은 작업(프로젝트+카테고리+제목이 동일)을 여러 번 수행(세션 분리)하여 각각 다른 비고를 남긴 경우의 처리 방식이 개선되었습니다.

- **기존**: 가장 마지막에 남긴 비고 1개만 시간관리에 덮어쓰기 형태로 반영되었습니다.
- **변경**: 각각의 세션에 작성했던 비고가 텍스트 줄바꿈 형태로 모두 묶여서 반영됩니다. 동일한 내용의 비고가 여러 번 작성된 경우, 중복을 제거하고 보여줍니다.

## 3. 주간일정 토글 버그 수정

- 주간일정 화면의 상단 필터 버튼 "전체보기 / 관리업무 제외"와 "특정 프로젝트 제외" 셀렉트 박스의 표시 상태가 반대로 작동하는 버그를 정상적으로 수정하였습니다.
