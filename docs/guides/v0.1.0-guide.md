## 1. 시작하기

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 시스템 요구사항

- 최신 브라우저 (Chrome, Firefox, Safari, Edge)
- JavaScript 활성화 필요
- 권장 해상도: 1280px 이상

## 2. 화면 구성

### 네비게이션 메뉴

| 메뉴 | 설명 |
|------|------|
| 일간 타이머 | 오늘의 작업 기록 및 타이머 |
| 주간 일정 | 주간 업무 현황 리포트 |
| 건의사항 | 피드백 게시판 (준비 중) |
| 설정 | 테마, 환경 설정 |

### 일간 타이머 레이아웃 (3단 구성)

```
┌─────────────────────────────────────────────────────┐
│                    네비게이션 바                      │
├───────────┬─────────────────────────────────────────┤
│           │  ■ 활성 타이머 (현재 진행 중인 작업)      │
│  작업     │  ■ 타이머 입력 (새 작업 시작)            │
│  프리셋   ├─────────────────────────────────────────┤
│  패널     │  ■ 간트 차트 (타임라인 시각화)           │
│           ├─────────────────────────────────────────┤
│           │  ■ 작업 기록 리스트                      │
└───────────┴─────────────────────────────────────────┘
```

## 3. 일간 타이머

### 3.1 새 작업 시작하기

1. **타이머 입력창**에 작업 제목 입력
2. (선택) 게시판 번호 입력
3. (선택) 카테고리 선택
4. **Enter** 키 또는 ▶ 버튼 클릭

### 3.2 작업 프리셋 사용하기

왼쪽 **작업 프리셋** 패널에서:
- 최근 사용한 작업이 자동으로 표시됨
- 작업 클릭 시 바로 타이머 시작
- ☆ 버튼으로 즐겨찾기 (준비 중)

### 3.3 타이머 제어

| 버튼 | 기능 |
|------|------|
| 일시정지 | 작업 일시 중단 |
| 재개 | 일시정지 해제 |
| 완료 | 작업 완료 및 기록 저장 |
| ⬛ (중단) | 작업 삭제 (기록 안 함) |

### 3.4 간트 차트 (타임라인)

- 06:00 ~ 익일 06:00 시간축
- 작업 막대에 마우스 오버 시 상세 정보 표시
- **빈 공간 드래그**로 수동 작업 기록 가능
- 빨간 세로선 = 현재 시간

### 3.5 작업 기록 리스트

- 최근 업무 기록 목록
- **완료된 항목 보기** 토글로 완료 작업 표시/숨김
- 각 항목에서:
  - ✏️ 수정
  - ▶ 재시작 (동일 작업으로 새 타이머)
  - 🗑️ 삭제

## 4. 주간 일정

### 4.1 기간 필터

| 옵션 | 설명 |
|------|------|
| 오늘 | 당일 기록만 |
| 이번 주 | 이번 주 월~일 |
| 이번 달 | 이번 달 전체 |
| 직접 지정 | 시작일 ~ 종료일 선택 |

### 4.2 데이터 확인

- 게시판 번호별 그룹화
- 총 소요 시간 표시
- 포함된 업무 요약

### 4.3 CSV 내보내기

**CSV 다운로드** 버튼 클릭 시:
- 엑셀 호환 형식 (한글 지원)
- 파일명: `time_report_YYYY-MM-DD.csv`

## 5. 설정

### 5.1 테마 커스터마이징

**테마 프리셋**:
- 기본 (검정)
- 초록, 보라, 빨강, 주황, 청록, 파랑

**커스텀 색상**:
- HEX 값 직접 입력 (예: `#3b82f6`)
- 상단 바 및 주요 버튼에 적용

### 5.2 업무 환경

- **점심시간 설정**: 시작/종료 시간
- **작업명 자동 완성**: ON/OFF

### 5.3 데이터 관리

| 기능 | 설명 |
|------|------|
| 데이터 내보내기 | JSON 파일로 백업 |
| 데이터 가져오기 | JSON 파일에서 복원 (기존 데이터 덮어씀) |

## 7. FAQ

### Q: 데이터는 어디에 저장되나요?
**A:** 브라우저의 LocalStorage에 저장됩니다. 브라우저 데이터를 삭제하면 기록이 사라질 수 있으니 정기적으로 백업하세요.

### Q: 다른 기기에서 사용할 수 있나요?
**A:** 현재 버전은 로컬 저장만 지원합니다. 설정 > 데이터 내보내기/가져오기로 수동 동기화 가능합니다.

### Q: 모바일에서 사용 가능한가요?
**A:** 반응형 디자인으로 모바일 브라우저에서 사용 가능합니다. PWA 기능은 향후 추가 예정입니다.

### Q: 시간이 정확하지 않아요
**A:** 브라우저 탭이 비활성화되면 타이머가 1분 단위로만 업데이트됩니다. 탭을 다시 활성화하면 즉시 동기화됩니다.
