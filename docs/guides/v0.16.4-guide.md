## 1. 시작하기

v0.16.4는 화면 크기 조절 기능의 조작성을 개선한 패치입니다.

### 기존 사용자
- 별도의 조치가 필요 없습니다.
- 기존에 50% 이하로 설정된 경우, 설정 페이지에서 프리셋 버튼을 클릭하여 원하는 크기로 변경해주세요.

### 새 사용자
- 변경 사항 없이 바로 사용할 수 있습니다.

## 2. 주요 변경사항

### 화면 크기 조절 개선
- 프리셋 버튼이 추가되어 슬라이더를 드래그하지 않고도 빠르게 화면 크기를 변경할 수 있습니다.
- 슬라이더 범위가 70%~150%로 조정되어 극단적인 줌 레벨로 인한 조작 불가 문제를 방지합니다.
- 100%가 아닌 경우 "100%로 되돌리기" 버튼이 표시되어 빠르게 기본 크기로 복원할 수 있습니다.

## 3. 화면 크기 조절

### 프리셋 버튼 사용
설정 페이지의 "화면 크기" 섹션에서 다음 프리셋 버튼을 클릭하여 즉시 변경할 수 있습니다:
- **작게 (80%)**: 더 많은 콘텐츠를 한 화면에 표시
- **약간 작게 (90%)**: 약간 콤팩트한 화면
- **보통 (100%)**: 기본 크기
- **약간 크게 (110%)**: 약간 확대된 화면
- **크게 (120%)**: 큰 글씨와 요소로 표시

### 슬라이더 사용
프리셋 이외의 세밀한 조절은 슬라이더를 드래그하여 5% 단위로 조절할 수 있습니다 (70%~150%).

### 기본 크기 복원
- "100%로 되돌리기" 버튼을 클릭하거나, 페이지 하단의 "기본값 복원" 버튼을 사용하세요.

## 4. FAQ

### Q: 이전에 50%로 설정했는데 화면이 너무 작아서 조작이 안 됩니다.
A: 브라우저의 개발자 도구(F12) → Console에서 `document.documentElement.style.zoom = '1'`을 입력하거나, 브라우저 캐시를 삭제하면 기본값으로 돌아옵니다. 또는 설정 페이지의 "보통 (100%)" 프리셋 버튼을 클릭하세요.

### Q: 슬라이더 범위가 왜 70%~150%로 제한되나요?
A: CSS zoom이 극단적인 값(50% 이하)일 때 브라우저의 포인터 이벤트 좌표 계산에 오류가 생겨 UI 조작이 불가능해질 수 있기 때문입니다. 프리셋 버튼을 통해 빠르게 변경할 수 있도록 개선했습니다.
