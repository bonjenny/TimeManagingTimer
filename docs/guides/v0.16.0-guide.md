## 1. 시간 표시 모드 토글

주간 일정에서 "누적시간"과 "당일시간" 중 원하는 시간 표시 방식을 선택할 수 있습니다.

### 누적시간 (기본값)
- 작업명 기준으로 처음 시작한 날부터 해당 날짜까지의 전체 누적 시간
- 여러 날에 걸쳐 진행한 작업의 총 투입 시간을 확인할 때 유용

### 당일시간
- 해당 날짜에 실제로 작업한 시간만 표시
- 하루 동안의 작업 배분과 시간 사용을 확인할 때 유용

## 2. 주간 일정에서 사용하기

1. **주간 일정** 탭으로 이동합니다
2. 하단의 **복사 미리보기** 영역에서 "누적시간 / 당일시간" 토글 버튼을 찾습니다
3. 원하는 모드를 클릭합니다
4. 주간일정 카드와 복사 미리보기의 시간 표시가 즉시 변경됩니다

### 적용 범위
| 영역 | 반영 여부 |
|------|-----------|
| 프로젝트 헤더 (시간 표시) | O |
| 하위 업무 목록 (시간 표시) | O |
| 복사 미리보기 (간단형/상세형/라벨형) | O |
| 클립보드 복사 결과 | O |

## 3. 복사 시 반영

복사 버튼을 클릭하면, 현재 선택된 시간 표시 모드에 따라 복사 내용이 달라집니다.

### 누적시간 모드 복사 예시
```
2/24 (월)
[A26_00001] 프로젝트명  (진행상태: 진행중, 시작일자: 2/10(월), 누적시간: 45:30)
  > 기능 개발 (누적시간: 30:00)
  > 테스트 작성 (누적시간: 15:30)
```

### 당일시간 모드 복사 예시
```
2/24 (월)
[A26_00001] 프로젝트명  (진행상태: 진행중, 시작일자: 2/10(월), 당일시간: 03:00)
  > 기능 개발 (당일시간: 02:00)
  > 테스트 작성 (당일시간: 01:00)
```

## 4. FAQ

### Q: 선택한 모드가 다음에도 유지되나요?
A: 네, 선택한 시간 표시 모드는 브라우저에 저장되어 다음 방문 시에도 유지됩니다.

### Q: 일간 타이머의 "오늘시간"과 주간 일정의 "당일시간"은 같은 건가요?
A: 계산 방식이 약간 다릅니다. 일간 타이머의 "오늘시간"은 06:00~익일 06:00 기준이고, 주간 일정의 "당일시간"은 00:00~익일 00:00 기준입니다. 대부분의 경우 동일한 값이지만, 새벽 시간대 작업이 있으면 차이가 발생할 수 있습니다.

### Q: 누적시간과 당일시간의 차이가 큰 이유는?
A: 누적시간은 해당 작업(작업명 기준)의 전체 투입 시간이므로, 여러 날에 걸쳐 진행한 작업일수록 누적시간이 크게 나타납니다. 당일시간은 해당 날짜의 작업 시간만 보여줍니다.
