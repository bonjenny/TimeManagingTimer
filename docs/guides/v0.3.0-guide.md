## 2. 즐겨찾기 사용법

### 즐겨찾기 추가
1. 프리셋 목록에서 원하는 항목을 찾습니다.
2. 항목 우측의 **별(☆)** 아이콘을 클릭합니다.
3. 아이콘이 **노란 별(⭐)**로 변경됩니다.

### 즐겨찾기 해제
1. 즐겨찾기된 항목(노란 별)을 클릭합니다.
2. 아이콘이 빈 별(☆)로 변경됩니다.

### 특징
- 즐겨찾기한 프리셋은 **목록 상단에 고정**됩니다.
- 즐겨찾기는 **브라우저를 닫아도 유지**됩니다.
- 제목 기준으로 즐겨찾기를 관리합니다.

## 3. 프리셋 수동 추가

기록 없이도 자주 사용하는 작업을 미리 등록할 수 있습니다.

### 추가 방법
1. 프리셋 패널 상단의 **+ 버튼**을 클릭합니다.
2. "새 프리셋 추가" 모달이 나타납니다.
3. 정보를 입력합니다:
   - **작업 제목** (필수): 프리셋 이름
   - **게시판 번호** (선택): 프로젝트/게시판 번호
   - **카테고리** (선택): 업무 분류
   - **프리셋 색상** (선택): 시각적 구분용 색상
4. **추가** 버튼을 클릭합니다.

### 수동 프리셋 표시
- 수동으로 추가한 프리셋은 **"수동"** 라벨이 표시됩니다.
- 자동 생성된 프리셋과 구분됩니다.

---

## 4. 프리셋 색상 커스텀

수동 프리셋에 고유 색상을 지정하여 시각적으로 구분할 수 있습니다.

### 색상 옵션

| 색상 | 코드 |
|------|------|
| 기본 (없음) | - |
| 파랑 | #3b82f6 |
| 초록 | #10b981 |
| 보라 | #8b5cf6 |
| 빨강 | #ef4444 |
| 주황 | #f97316 |
| 청록 | #06b6d4 |
| 노랑 | #eab308 |

### 색상 표시
- 색상이 지정된 프리셋은 **좌측에 색상 바**가 표시됩니다.
- 프리셋 클릭 시 해당 색상 정보는 타이머에 전달되지 않습니다. (시각적 구분용)

---

## 5. 프리셋 수정/삭제

### 수정
1. 수동 프리셋의 **연필(✏️) 아이콘**을 클릭합니다.
2. "프리셋 수정" 모달에서 정보를 변경합니다.
3. **수정** 버튼을 클릭합니다.

### 삭제
1. 수동 프리셋의 연필 아이콘을 클릭합니다.
2. 모달 좌측 하단의 **삭제** 버튼을 클릭합니다.
3. 프리셋이 즉시 삭제됩니다.

### 참고
- **자동 생성된 프리셋**은 수정/삭제할 수 없습니다.
- 자동 프리셋은 기록에서 자동으로 추가/제거됩니다.
- 즐겨찾기는 수동/자동 프리셋 모두 가능합니다.

## 데이터 저장 위치

프리셋 관련 데이터는 브라우저 LocalStorage에 저장됩니다:

| 데이터 | 저장 키 |
|--------|---------|
| 즐겨찾기 목록 | `timekeeper-preset-favorites` |
| 수동 프리셋 | `timekeeper-manual-presets` |

브라우저 캐시 삭제 시 이 데이터도 함께 삭제될 수 있으니 주의하세요.
