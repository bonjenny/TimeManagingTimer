## 1. 시작하기

v0.21.1에서는 최근 업무 기록의 상세내역(작업 이력)에 정렬 기능이 추가되었습니다. 업무 그룹을 펼쳤을 때 세션들을 오래된 순 또는 최근 순으로 정렬하여 볼 수 있습니다.

## 2. 최근 업무 기록 세션 정렬 기능

### 개요
업무 그룹의 상세내역을 펼쳤을 때, 작업 이력 헤더 우측에 정렬 버튼이 표시됩니다. 이 버튼을 클릭하여 세션의 표시 순서를 변경할 수 있습니다.

### 정렬 옵션
- **오래된 순 (asc)**: 첫 번째 세션부터 마지막 세션까지 표시 (1회 → 2회 → 3회)
- **최근 순 (desc)**: 마지막 세션부터 첫 번째 세션까지 표시 (3회 → 2회 → 1회)

### 주요 기능
- 정렬 버튼 클릭으로 간편한 정렬 전환
- 시각적 피드백: 정렬 방향에 따라 아이콘 변경
  - ▼ (ArrowDownward): 오래된 순 (클릭 시 최근 순으로 변경)
  - ▲ (ArrowUpward): 최근 순 (클릭 시 오래된 순으로 변경)
- 사용자 선택 저장: localStorage에 정렬 설정이 저장되어 다음 방문 시에도 유지
- 세션 번호(#) 자동 조정: 정렬 방향에 맞게 번호 표시

## 3. 정렬 사용 방법

### 기본 사용법

1. **업무 그룹 펼치기**
   - 최근 업무 기록에서 업무 그룹을 클릭하여 펼칩니다
   - 상세내역이 표시되고 "작업 이력 (총 N회 작업)" 헤더가 보입니다

2. **정렬 버튼 클릭**
   - 헤더 우측의 정렬 아이콘 버튼을 클릭합니다
   - 아이콘이 변경되며 세션 순서가 변경됩니다

3. **정렬 상태 확인**
   - ▼ 아이콘: 오래된 순으로 정렬됨 (1회부터 표시)
   - ▲ 아이콘: 최근 순으로 정렬됨 (마지막 회차부터 표시)

### 사용 예시

#### 오래된 순에서 최근 순으로 변경
```
초기 상태 (▼ 아이콘):
#1 | 2026-03-18 | 09:00 ~ 10:30 | 90분
#2 | 2026-03-18 | 14:00 ~ 15:00 | 60분
#3 | 2026-03-19 | 10:00 ~ 11:00 | 60분

버튼 클릭 후 (▲ 아이콘):
#3 | 2026-03-19 | 10:00 ~ 11:00 | 60분
#2 | 2026-03-18 | 14:00 ~ 15:00 | 60분
#1 | 2026-03-18 | 09:00 ~ 10:30 | 90분
```

## 4. 정렬 동작 설명

### 정렬 기준
- 세션의 **시작 시간**을 기준으로 정렬됩니다
- 오래된 순: 시작 시간 오름차순 정렬
- 최근 순: 시작 시간 내림차순 정렬

### 세션 번호(#) 표시
- **오래된 순**: 1, 2, 3, ... (첫 세션이 1번)
- **최근 순**: N, N-1, N-2, ... (마지막 세션이 N번)
- 번호는 정렬 방향과 관계없이 세션의 실제 순서를 나타냅니다

### 정렬 상태 유지
- 정렬 설정은 localStorage에 저장됩니다
- 페이지를 새로고침하거나 다시 방문해도 설정이 유지됩니다
- 모든 업무 그룹에 동일한 정렬 설정이 적용됩니다

### 다른 기능과의 호환성
- 세션 추가/수정/삭제 시에도 정렬 상태가 유지됩니다
- 인라인 편집 기능과 정상적으로 동작합니다
- 날짜 변경 시에도 정렬 설정은 그대로 유지됩니다

## 5. FAQ

**Q: 정렬 설정이 모든 업무 그룹에 적용되나요?**
A: 네, 정렬 설정은 전역으로 적용됩니다. 모든 업무 그룹의 상세내역이 동일한 정렬 방향으로 표시됩니다.

**Q: 정렬 후 새 세션을 추가하면 어떻게 되나요?**
A: 새 세션은 정렬 순서에 맞는 위치에 표시됩니다. 최근 순 정렬 중이라면 새 세션이 맨 위에 표시됩니다.

**Q: 세션 번호(#)는 어떻게 결정되나요?**
A: 세션 번호는 세션의 실제 시간 순서를 나타냅니다. 오래된 순 정렬에서는 1부터 시작하고, 최근 순 정렬에서는 전체 세션 수부터 시작합니다.

**Q: 정렬 설정을 초기화할 수 있나요?**
A: 정렬 버튼을 클릭하여 원하는 방향으로 다시 설정하거나, 브라우저의 localStorage를 삭제하면 기본값(오래된 순)으로 초기화됩니다.

**Q: 진행 중인 작업의 세션도 정렬되나요?**
A: 네, 진행 중인 작업도 시작 시간 기준으로 정렬됩니다. 다만 종료 시간은 "진행 중"으로 표시됩니다.

**Q: 정렬 아이콘이 보이지 않아요.**
A: 업무 그룹을 펼쳐야 정렬 버튼이 표시됩니다. "작업 이력" 헤더 우측을 확인하세요.

**Q: 날짜를 변경하면 정렬이 초기화되나요?**
A: 아니요, 정렬 설정은 localStorage에 저장되어 날짜 변경과 관계없이 유지됩니다.

**Q: 여러 업무 그룹을 동시에 펼쳤을 때도 동일한 정렬이 적용되나요?**
A: 네, 모든 업무 그룹에 동일한 정렬 설정이 적용됩니다. 한 곳에서 정렬을 변경하면 펼쳐진 모든 그룹의 정렬이 함께 변경됩니다.

**Q: 세션 시간을 수정하면 정렬 순서가 바뀌나요?**
A: 세션 시간을 수정해도 현재 표시된 순서는 즉시 바뀌지 않습니다. 상세내역을 접었다가 다시 펼치면 수정된 시간 기준으로 재정렬됩니다.

**Q: 엑셀 Export 시 정렬 순서가 적용되나요?**
A: 상세내역의 정렬은 화면 표시용이며, 엑셀 Export에는 영향을 주지 않습니다. Export 파일은 시간 순서대로 생성됩니다.
