## 2. 주간 단위 이동

### 탐색 방법

상단의 주간 선택 UI를 사용합니다:

```
[ < ]  2026년 1/20 ~ 1/26  [ > ]  [이번주]
```

| 버튼 | 동작 |
|------|------|
| `<` | 이전 주로 이동 |
| `>` | 다음 주로 이동 |
| 이번 주 | 현재 주로 복귀 (이번 주가 아닐 때만 표시) |

### 표시 규칙

- **이번 주**: 검정 배경, 흰색 글씨
- **다른 주**: 흰색 배경, 검정 글씨
- 주 범위: 월요일 ~ 일요일

## 3. 복사 템플릿

주간 업무 내용을 텍스트로 복사하여 보고서에 활용할 수 있습니다.

### 사용 방법

1. "템플릿 복사" 버튼을 클릭합니다.
2. 드롭다운에서 형식을 선택합니다:
   - **상세형 템플릿**: 게시판별 업무 목록 포함
   - **요약형 템플릿**: 게시판별 시간만 표시
3. 클립보드에 복사되었다는 알림이 표시됩니다.
4. 원하는 곳에 붙여넣기 합니다 (Ctrl+V).

### 상세형 템플릿 예시

```
📊 주간 업무 보고서 (2026년 1/20 ~ 1/26)

⏱️ 총 업무 시간: 40시간 30분

📋 게시판별 상세 내역
────────────────────────────────────────

[#12345] 15시간 20분 (8건)
  • 사용자 인증 모듈 개발
  • API 연동 테스트
  • 버그 수정

[#67890] 10시간 45분 (5건)
  • 데이터베이스 마이그레이션
```

### 요약형 템플릿 예시

```
주간 업무 요약 (2026년 1/20 ~ 1/26)

총 시간: 40시간 30분

• #12345: 15시간 20분 (8건)
• #67890: 10시간 45분 (5건)
• 기타: 5시간 (3건)
```

## 4. 카테고리별 시각화

주간 업무 시간을 카테고리별로 막대 차트로 표시합니다.

### 차트 읽는 법

- 막대 길이: 해당 카테고리의 업무 시간 비중
- 색상: 카테고리별 고유 색상
- 정렬: 시간이 많은 순서

### 카테고리 색상

| 카테고리 | 색상 |
|----------|------|
| 분석 | 파랑 |
| 개발 | 초록 |
| 개발자테스트 | 보라 |
| 테스트오류수정 | 빨강 |
| 센터오류수정 | 주황 |
| 환경세팅 | 청록 |
| 회의 | 노랑 |
| 기타 | 회색 |

## 5. 주간 보고서 작성 팁

### 1. CSV 다운로드 활용

1. "CSV 다운로드" 버튼을 클릭합니다.
2. 엑셀에서 파일을 엽니다.
3. 피벗 테이블로 분석하거나 보고서 양식에 붙여넣습니다.

### 2. 템플릿 복사 활용

1. **이메일/메신저**: 요약형 템플릿 사용
2. **주간 보고서**: 상세형 템플릿 사용
3. 필요에 따라 텍스트 수정 후 제출

### 3. 과거 주 조회

- 이전 주로 이동하여 지난 기록 확인
- 월간 보고서 작성 시 각 주별 데이터 수집
