# TimeKeeper v0.14.1 사용설명서

v0.14.0 사용설명서를 기준으로 하며, v0.14.1에서 변경·추가된 내용만 정리합니다.

## v0.14.1 변경 사항

### 배포 캘린더

- **기본 표시**: 열면 **저번주~이번주** 2주가 보입니다 (오늘 기준).
- **주차 표시**: 상단에 예) "2026년 2월 1주차 ~ 2주차" 형식으로 표시됩니다.
- **이번 주로 이동**: 버튼 클릭 시 저번주~이번주 뷰로 이동합니다.
- **열 너비**: 날짜 헤더와 셀 너비가 맞고, 긴 프로젝트명은 말줄임표(...)로 표시됩니다.
- **다크모드**: 잡 상태 글씨가 검정으로 고정되어 밝은 배경색에서도 잘 보입니다.
- **잡 색상**: 새 잡 기본 색상과 잡 색상 설정 선택지는 **설정 > 컬러 팔레트** 테마를 사용합니다.

### 전역 표시

- **배경색**: 라이트/다크 모드 모두 **body**에 배경색이 적용됩니다 (다른 div가 아닌 body 기준).
- 라이트: `#fafafa`, 다크: `#121212`

## 단축키 (요약)

| 단축키 | 기능 |
|--------|------|
| `Alt + 3` | 배포 캘린더 |
| `Alt + 1` | 일간 타이머 |
| `Alt + 2` | 주간 일정 |
| `Alt + T` | 오늘로 이동 |
| `F8` | 새 작업 추가 |

전체 기능은 [v0.14.0 사용설명서](v0.14.0-guide.md)를 참고하세요.
