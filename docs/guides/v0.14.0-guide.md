## 1. 시작하기

TimeKeeper는 업무 시간을 기록하고 관리하는 도구입니다. v0.14.0에서는 배포 일정을 관리할 수 있는 월간 캘린더 기능이 추가되었습니다.

## 2. 주요 기능

### 타이머 기능
- 업무 시작/정지/완료
- 프로젝트 코드 및 카테고리 지정
- 간트 차트를 통한 시각적 표시

### 업무 기록
- 일자별 업무 기록 조회
- 업무 그룹화 및 세션 관리
- 휴지통 기능

### 주간 일정
- 주간 단위 업무 요약
- 프로젝트별 누적 시간 표시

## 3. 새로운 기능: 배포 캘린더

### 3.1 배포 캘린더란?

배포 일정을 월간 캘린더 형태로 관리하고, HTML 테이블로 복사하여 웹에디터에 붙여넣을 수 있는 기능입니다.

### 3.2 배포 캘린더 접근 방법

- **탭 클릭**: 상단 메뉴에서 "배포 캘린더" 탭 클릭
- **단축키**: `Alt + 3`

### 3.3 캘린더 UI 구성

```
+--------------------------------------------------------------+
|  [<]  2026년 1월  [>]           [잡 색상 설정]  [HTML 복사]   |
+--------------------------------------------------------------+
| 1/26(월)      | 1/27(화)      | 1/28(수)      | 1/29(목)      | 1/30(금)      |
+---------------+---------------+---------------+---------------+---------------+
|               | [다크모드     | [다크모드     | 연차          |               |
|               |  스테이지]    |  수시]        |               |               |
+---------------+---------------+---------------+---------------+---------------+
```

- **월~금 표시**: 평일 5일 기준으로 표시
- **2주 단위**: 한 화면에 2주 분량 표시
- **네비게이션**: 좌우 화살표로 주 이동, 오늘 아이콘으로 현재 주 이동

### 3.4 이벤트 추가/수정

1. **빈 셀 클릭**: 해당 날짜에 새 이벤트 추가
2. **기존 이벤트 클릭**: 해당 이벤트 수정

### 이벤트 정보

| 필드 | 설명 | 예시 |
|------|------|------|
| **프로젝트 코드** | 기존 등록된 프로젝트 코드 선택 | A25_07788 |
| **표시명** | 캘린더에 표시될 이름 | HTML 다크모드 |
| **진행상태** | 배포 단계 | 스테이지, 수시, 테스트 등 |
| **휴일** | 휴일 여부 (빨간색으로 표시) | 신정, 연차 등 |

### 3.5 잡 색상 설정

1. **"잡 색상 설정" 버튼 클릭**: 색상 설정 다이얼로그 열기
2. **잡 목록 확인**: 등록된 이벤트의 프로젝트 코드 목록 표시
3. **색상 선택**: 
   - 프리셋 색상 12가지 중 선택
   - 또는 컬러피커로 커스텀 색상 지정

### 3.6 HTML 테이블 복사

1. **"HTML 복사" 버튼 클릭**: 현재 표시된 2주 캘린더를 HTML 테이블로 복사
2. **웹에디터에 붙여넣기**: 
   - ecount 웹에디터
   - Notion
   - Excel
   - Confluence 등

### 복사된 HTML 특징

- 날짜 헤더: 베이지색 배경, 볼드 텍스트
- 이벤트: 잡별 배경색 적용
- 휴일: 빨간색 텍스트, 가운데 정렬

## 4. FAQ

### Q: 이벤트를 삭제하려면 어떻게 하나요?
A: 해당 이벤트를 클릭하여 수정 모달을 열고, 우측 상단의 삭제(휴지통) 아이콘을 클릭합니다.

### Q: 프로젝트 코드가 자동완성에 나오지 않아요
A: 프로젝트 코드는 일간 타이머에서 입력한 기록이 있어야 자동완성 목록에 표시됩니다. 수동으로 코드를 입력해도 됩니다.

### Q: HTML 복사가 안 돼요
A: 브라우저의 클립보드 권한을 확인하세요. 일부 브라우저에서는 권한 허용이 필요합니다.

### Q: 색상이 저장되지 않아요
A: 모든 데이터는 localStorage에 저장됩니다. 시크릿 모드에서는 데이터가 유지되지 않습니다.

### Q: 3주 이상 표시할 수 있나요?
A: 현재는 2주 단위로 고정되어 있습니다. 네비게이션 버튼으로 다른 주간으로 이동할 수 있습니다.
