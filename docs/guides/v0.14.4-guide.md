# TimeKeeper v0.14.4 사용설명서

v0.14.3 사용설명서를 기준으로 하며, v0.14.4에서 변경된 내용만 정리합니다.

## v0.14.4 변경 사항

### 간트 차트 > 드래그 앤 드롭

- **행 인덱스 수정**: 이전에는 7번째 줄 이후에서 타임라인을 드래그할 때 세션이 한 줄 아래 행에 추가되는 현상이 있었습니다. v0.14.4부터는 **클릭한 행**을 기준으로 세션 추가 모달이 열리며, 해당 행의 작업 정보가 올바르게 채워집니다.

## 단축키 (요약)

| 단축키 | 기능 |
|--------|------|
| `Alt + 3` | 배포 캘린더 |
| `Alt + 1` | 일간 타이머 |
| `Alt + 2` | 주간 일정 |
| `Alt + T` | 오늘로 이동 |
| `F8` | 새 작업 추가 |

전체 기능은 [v0.14.3 사용설명서](v0.14.3-guide.md)를 참고하세요.
