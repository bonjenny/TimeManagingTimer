## 1. 시작하기

v0.20.0에서는 미래 시간에 대한 일정을 사전 등록하고, 해당 시간이 되면 자동으로 타이머가 전환되는 기능이 추가되었습니다.

## 2. 예약 일정 기능

### 개요
- 미래 시간대의 작업(예: 4시~5시 회의)을 미리 등록
- 예약 시간이 되면 현재 진행중인 타이머가 자동으로 paused 처리
- 예약된 작업의 타이머가 자동으로 시작
- 예약된 종료 시간에 자동으로 완료 처리

### 자동 스탑은 항상 활성화
- 별도의 on/off 설정 없이 항상 동작합니다

### 등록 방법 3가지
| 방법 | 동작 |
|------|------|
| 간트차트 좌클릭 드래그 | 미래 시간이면 자동으로 SCHEDULED, 과거 시간이면 COMPLETED |
| 간트차트 우클릭 드래그 | 항상 SCHEDULED로 생성 (주황색 드래그 영역으로 표시) |
| TimerInput 예약 모드 | 시계 아이콘 클릭 후 시간 설정하여 등록 |

## 3. 간트차트에서 예약 등록

### 좌클릭 드래그 (미래 시간 자동 감지)
1. 간트차트에서 미래 시간 영역을 **좌클릭**으로 드래그합니다
2. 작업 정보를 입력하는 모달이 표시됩니다
3. 제목, 프로젝트, 카테고리를 입력하고 저장합니다
4. 미래 시간이 감지되면 자동으로 SCHEDULED 상태로 생성됩니다

### 우클릭 드래그 (강제 예약)
1. 간트차트에서 원하는 시간 영역을 **우클릭**으로 드래그합니다
2. 드래그 중 주황색 점선 영역으로 표시됩니다
3. 모달 타이틀이 "예약 일정 등록"으로 표시됩니다
4. 시간대와 관계없이 항상 SCHEDULED 상태로 생성됩니다

### 시각적 구분
- 예약 작업은 간트차트에서 **점선 테두리 + 사선 패턴**으로 표시됩니다
- 툴팁에 **[예약]** 라벨이 표시됩니다

## 4. TimerInput에서 예약 등록

1. "무엇을 하고 계신가요?" 입력창 옆의 **시계 아이콘 버튼**(ScheduleIcon)을 클릭합니다
2. 예약 모드가 활성화되면 버튼이 주황색으로 변합니다
3. 하단에 시작 시간과 종료 시간 입력 필드가 나타납니다
   - 기본값: 다음 정시 ~ +1시간 (예: 현재 14:30이면 15:00~16:00)
4. 작업명, 프로젝트, 카테고리 등을 입력합니다
5. Enter 또는 달력 아이콘 버튼(EventNoteIcon)을 클릭하여 예약을 등록합니다

## 5. 자동 타이머 전환

### 예약 시간 도래 시
- 5초 간격으로 예약 작업을 확인합니다
- 예약 시작 시간이 되면:
  - 현재 진행중인 타이머가 자동으로 paused 처리됩니다
  - 예약된 작업의 타이머가 자동으로 시작됩니다

### 예약 종료 시간 도래 시
- 예약된 종료 시간에 자동으로 COMPLETED 처리됩니다

### 앱 재시작 시
- 이미 시작 시간이 지났지만 종료 시간이 안 지난 예약: 즉시 시작
- 시작/종료 시간이 모두 지난 예약: 자동으로 COMPLETED 처리

## 6. 예약 작업 관리

### 수정
- 간트차트에서 예약 작업을 더블클릭하여 수정할 수 있습니다
- 시작/종료 시간, 제목, 프로젝트 등을 변경할 수 있습니다

### 삭제
- 간트차트에서 예약 작업을 우클릭하고 '삭제'를 선택합니다

### 즉시 시작
- 간트차트에서 예약 작업을 우클릭하고 '지금 시작'을 선택하면 즉시 타이머가 시작됩니다

### 시간 조정
- 간트차트에서 예약 작업의 양끝을 드래그하여 시간을 조정할 수 있습니다

## 7. FAQ

**Q: 예약 작업은 최근 업무 기록에 표시되나요?**
A: 아니요. 예약 작업은 간트차트에서만 표시됩니다. 실제로 시작된 후에 업무 기록에 나타납니다.

**Q: 예약 작업이 주간 일정이나 시간관리 표에 포함되나요?**
A: 아니요. 예약 상태의 작업은 주간 일정과 시간관리 표에서 제외됩니다. 완료된 후에만 포함됩니다.

**Q: 여러 예약을 동시에 등록할 수 있나요?**
A: 네. 여러 시간대에 예약을 등록할 수 있습니다. 시간순으로 자동 처리됩니다.

**Q: 회의가 예상보다 길어지면 어떻게 하나요?**
A: 예약된 종료 시간에 자동 완료됩니다. 이후 수동으로 타이머를 다시 시작하거나, 미리 간트차트에서 종료 시간을 조정할 수 있습니다.

**Q: 좌클릭 드래그와 우클릭 드래그의 차이는?**
A: 좌클릭 드래그는 과거 시간이면 COMPLETED, 미래 시간이면 SCHEDULED로 생성됩니다. 우클릭 드래그는 시간대와 관계없이 항상 SCHEDULED로 생성됩니다.
