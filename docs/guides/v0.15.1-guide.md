# TimeKeeper v0.15.1 사용설명서

## 목차
1. 변경 사항
2. 일간 타이머 시간 표시

## 1. 변경 사항

v0.15.1은 일간 타이머의 누적시간 표시 오류를 수정한 패치 버전입니다.

## 2. 일간 타이머 시간 표시

일간 타이머 목록에는 업무별로 두 가지 시간이 표시됩니다.

### 오늘시간
- 선택된 날짜 범위(06:00 ~ 익일 06:00) 내에서 해당 업무에 투입한 시간
- 상단 헤더의 "총" 합계도 이 값의 합산입니다

### 총시간 (누적시간)
- 해당 업무명으로 기록된 **전체 세션의 누적 시간** (선택된 날짜까지)
- 주간 일정의 누적시간과 동일한 방식으로 계산됩니다
- 여러 날에 걸쳐 작업한 업무의 전체 투입 시간을 확인할 수 있습니다

### 예시
| 오늘시간 | 총시간 |
|---------|--------|
| 30분 / 00:30 시간 | 150분 / 02:30 시간 |

위 예시에서 오늘은 30분 작업했지만, 이전 날짜들을 포함한 전체 누적 시간은 2시간 30분입니다.
