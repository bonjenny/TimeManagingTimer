## 2. F8 단축키로 새 작업 시작

### 사용 방법

1. 앱의 **어떤 화면에서든** `F8` 키를 누릅니다.
2. "새 작업 시작" 모달이 나타납니다.
3. 작업 제목을 입력합니다 (필수).
4. (선택) 게시판 번호, 카테고리를 입력합니다.
5. `Enter` 키 또는 "타이머 시작" 버튼을 클릭합니다.

### 자동 완성

- 작업 제목 입력 시 최근 30일간 기록된 작업명이 자동 완성됩니다.
- 드롭다운에서 선택하면 빠르게 입력할 수 있습니다.

### 팁

- 일간 타이머 페이지가 아닌 **주간 일정, 설정** 등 다른 페이지에서도 F8으로 작업을 시작할 수 있습니다.
- 모달이 열리면 제목 입력창에 자동으로 포커스됩니다.

## 3. 간트 차트에서 시간 수정

### 리사이즈 기능

**완료된 작업**의 막대 양 끝을 드래그하여 시작/종료 시간을 수정할 수 있습니다.

### 사용 방법

1. 간트 차트에서 **완료된 작업** 막대에 마우스를 올립니다.
2. 막대의 **좌측 끝** 또는 **우측 끝**에 커서를 가져갑니다.
3. 커서가 `↔` 모양으로 바뀌면 드래그합니다.
4. 원하는 위치에서 마우스를 놓습니다.
5. 변경된 시간이 즉시 저장됩니다.

### 제한 사항

- **진행 중인 작업**은 리사이즈할 수 없습니다. (완료 후 수정)
- 최소 작업 시간은 **5분**입니다.
- 시작 시간이 종료 시간보다 늦어질 수 없습니다.

### 시각적 피드백

- 드래그 중에는 막대가 실시간으로 늘어나거나 줄어듭니다.
- 리사이즈 핸들에 마우스를 올리면 흰색 표시가 나타납니다.

## 4. 날짜 이동

### 날짜 선택 UI

간트 차트 상단에 날짜 이동 컨트롤이 있습니다.

```
[ < ]  2026. 1. 26. (월)  [ > ]  [오늘]
```

| 버튼 | 기능 |
|------|------|
| `<` | 이전 날짜로 이동 |
| `>` | 다음 날짜로 이동 |
| 오늘 | 오늘 날짜로 돌아가기 (오늘이 아닐 때만 표시) |

### 날짜 표시

- **오늘**: 검정 배경에 흰색 글씨
- **다른 날**: 회색 배경

### 하루의 기준

- 하루는 **06:00 ~ 익일 06:00**으로 계산됩니다.
- 새벽 2시에 앱을 열면 전날 기록이 표시됩니다.

### 참고 사항

- 과거 날짜에서는 **현재 시간 표시선**(빨간 선)이 나타나지 않습니다.
- 과거 날짜에서는 **활성 타이머**가 표시되지 않습니다.
- 과거 날짜의 기록도 **리사이즈**로 시간 수정이 가능합니다.
