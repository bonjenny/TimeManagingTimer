## 1. 간트 차트 자동 조정 기능

v0.11.0부터 간트 차트에서 **동시에 여러 작업이 진행될 수 없습니다**. 시간이 겹치는 작업을 생성하거나 수정하려고 하면 자동으로 시간이 조정됩니다.

### 핵심 원칙

- 한 시점에 하나의 작업만 존재 가능
- 시간 충돌 발생 시 자동으로 밀어내기/당기기 적용
- 조정 불가능한 경우 작업 생성 취소

## 2. 드래그 시작점 스냅

기존 작업 행에서 새 세션을 추가할 때, 드래그 시작점이 자동으로 스냅됩니다.

### 사용 방법

1. 간트 차트에서 기존 작업이 있는 행을 찾습니다.
2. 해당 행의 빈 영역에서 드래그를 시작합니다.
3. 드래그 시작점이 **가장 가까운 세션 경계(±15분 이내)**에 자동으로 맞춰집니다.

### 활용 예시

| 상황 | 동작 |
|---|---|
| 이전 세션 종료 시간 근처에서 드래그 시작 | 이전 세션 종료 시간에 정확히 맞춤 |
| 다음 세션 시작 시간 근처에서 드래그 종료 | 다음 세션 시작 시간에 정확히 맞춤 |

이 기능으로 세션 간 빈틈 없이 연속적인 작업 기록이 가능합니다.

## 3. 시간 충돌 방지

### 자동 조정 동작

작업 생성 또는 리사이즈 시 기존 작업과 시간이 겹치면:

1. **시작점 충돌**: 새 작업의 시작 시간을 기존 작업 종료 시간으로 밀어냅니다.
2. **종료점 충돌**: 새 작업의 종료 시간을 기존 작업 시작 시간으로 당깁니다.
3. **완전 포함**: 새 작업을 기존 작업 앞으로 축소합니다.

### 알림 메시지

| 메시지 | 의미 |
|---|---|
| "시간 충돌로 인해 자동 조정되었습니다." | 충돌이 있었지만 자동으로 해결됨 |
| "유효한 시간 범위가 없습니다." | 조정 후에도 최소 5분 확보 불가, 작업 취소됨 |

### 최소 세션 길이

모든 작업 세션은 **최소 5분** 이상이어야 합니다. 자동 조정 후 5분 미만이 되면 작업 생성이 취소됩니다.

## 4. FAQ

### Q: 기존에 겹치는 작업이 있으면 어떻게 되나요?
A: 기존 데이터는 그대로 유지됩니다. 새로 생성하거나 수정하는 작업에만 자동 조정이 적용됩니다.

### Q: 스냅 기능을 끌 수 있나요?
A: 현재 버전에서는 스냅 기능이 항상 활성화되어 있습니다. ±15분 범위를 벗어나면 스냅되지 않습니다.

### Q: 리사이즈할 때도 자동 조정이 적용되나요?
A: 네, 리사이즈 시에도 동일한 자동 조정 로직이 적용됩니다. 기존 작업과 겹치지 않도록 자동으로 시간이 조정됩니다.

### Q: 완전히 겹치는 시간대에 작업을 추가하면?
A: 유효한 시간 범위가 없으므로 "유효한 시간 범위가 없습니다" 메시지와 함께 작업 생성이 취소됩니다.
