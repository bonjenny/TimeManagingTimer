## 1. 업데이트 요약

v0.11.0에서는 **시간 입력 UX**가 크게 개선되었습니다.

### 주요 변경사항

| 기능 | 설명 |
|---|---|
| **TimePicker** | 설정 페이지의 점심시간 입력에 시계 UI 적용 |
| **인라인 시간 편집** | 세션 테이블에서 시간 클릭 시 바로 수정 가능 |
| **충돌 자동 조정** | 시간 변경 시 이전/다음 세션과 충돌하면 자동 조정 |
| **모달 패딩 개선** | 건의사항 모달 내부 여백 반응형 적용 |

## 2. TimePicker로 시간 입력하기

### 2.1 점심시간 설정

1. **설정** 탭으로 이동
2. **업무 환경** 섹션에서 점심시간 입력 필드 클릭
3. 시계 아이콘이 있는 TimePicker가 열림
4. 시간을 선택하고 확인

### 2.2 TimePicker 사용법

| 동작 | 설명 |
|---|---|
| **필드 클릭** | TimePicker 팝업 열기 |
| **시간 선택** | 시/분 드래그 또는 클릭으로 선택 |
| **확인** | OK 버튼 또는 바깥 영역 클릭 |
| **취소** | Cancel 버튼 또는 Escape 키 |

## 3. 인라인 시간 편집

### 3.1 사용 방법

1. 일간 타이머 페이지에서 작업 항목 클릭 (세션 상세 펼치기)
2. **시작 시간** 또는 **종료 시간** 셀 클릭
3. 시간 입력 필드로 전환됨
4. 새로운 시간 입력 후:
   - **Enter**: 저장
   - **Escape**: 취소
   - **바깥 클릭**: 저장

### 3.2 충돌 자동 조정

시간을 변경했을 때 다른 세션과 겹치면 자동으로 조정됩니다.

#### 종료 시간 변경 시

| 상황 | 자동 조정 |
|---|---|
| 종료 시간 > 다음 세션 시작 시간 | 다음 세션의 시작 시간을 현재 종료 시간으로 변경 |
| 종료 시간 < 시작 시간 | 변경 무시 (유효하지 않음) |

#### 시작 시간 변경 시

| 상황 | 자동 조정 |
|---|---|
| 시작 시간 < 이전 세션 종료 시간 | 이전 세션의 종료 시간을 현재 시작 시간으로 변경 |
| 시작 시간 > 종료 시간 | 변경 무시 (유효하지 않음) |

### 3.3 예시

**변경 전:**
- 세션 1: 09:00 ~ 10:00
- 세션 2: 10:30 ~ 11:30

**세션 1의 종료 시간을 10:45로 변경:**
- 세션 1: 09:00 ~ 10:45
- 세션 2: 10:45 ~ 11:30 (시작 시간 자동 조정)

## 4. 자주 묻는 질문

### Q1. TimePicker가 안 열려요

브라우저를 새로고침(F5)해 주세요. 캐시가 오래된 경우 발생할 수 있습니다.

### Q2. 진행 중인 작업의 종료 시간을 수정할 수 있나요?

아니요, 진행 중인 작업(종료 시간이 없는)은 종료 시간을 편집할 수 없습니다. 작업을 완료한 후 수정하세요.

### Q3. 충돌 자동 조정을 끌 수 있나요?

현재 버전에서는 자동 조정을 끌 수 없습니다. 상세 수정(연필 아이콘)을 사용하면 개별 세션을 정확하게 수정할 수 있습니다.

### Q4. 시간을 잘못 수정했어요

Escape 키를 누르면 수정이 취소됩니다. 이미 저장된 경우 상세 수정 버튼을 눌러 다시 수정하세요.
