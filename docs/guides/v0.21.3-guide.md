# TimeKeeper v0.21.3 사용설명서

## 목차
1. 시작하기
2. 간트차트에서 비고 입력하기
3. FAQ

## 1. 시작하기

v0.21.3부터 **일간 타이머** 화면의 **간트차트**에서 업무를 드래그해 수동으로 만들거나, 막대를 우클릭·더블클릭해 수정할 때 **비고**를 입력할 수 있습니다.

## 2. 간트차트에서 비고 입력하기

### 새 업무 기록 (수동)
1. 간트차트 빈 영역을 드래그해 시간 범위를 지정합니다.
2. 열리는 **새 업무 기록 (수동)** (또는 예약 일정) 모달에서 업무 제목·프로젝트·카테고리와 함께 **비고**란에 메모를 입력합니다.
3. 저장하면 해당 세션 로그에 비고가 저장됩니다.

### 기존 작업에 세션 추가
1. 같은 제목 행에서 드래그해 세션을 추가하면, 기존 세션 중 하나에 적혀 있던 비고가 있으면 비고란에 미리 채워집니다. 필요하면 수정할 수 있습니다.

### 업무 기록 수정
1. 작업 막대를 **더블클릭**하거나 **우클릭 → 수정**을 선택합니다.
2. **업무 기록 수정** 모달의 **비고**에서 내용을 입력·수정합니다.
3. 저장하면 반영됩니다. 비고를 비우면 저장 시 비고가 제거됩니다.

비고는 **최근 업무 기록**, **시간관리** 등 다른 화면에서와 동일하게 `TimerLog.note`로 연동됩니다.

## 3. FAQ

**Q: 타이머 입력창의 비고와 같나요?**  
A: 네, 같은 필드입니다. 간트에서 저장한 비고는 목록·보내기 등에서 동일하게 보입니다.

**Q: 예약(SCHEDULED) 일정에도 비고가 붙나요?**  
A: 네, 수동 생성·수정 모달을 쓰는 경우와 동일하게 저장됩니다.
