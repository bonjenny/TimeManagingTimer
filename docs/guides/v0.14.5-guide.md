# TimeKeeper v0.14.5 사용설명서

## 목차
1. 주간일정 진행상태 입력 개선
2. 완료된 항목 보기 설정 유지
3. 완료 취소 동작 개선

## 1. 주간일정 진행상태 입력 개선

### 신규 상태 바로 추가
- 주간 일정 화면에서 프로젝트의 진행상태 드롭다운을 클릭하면 기존 상태 목록이 표시됩니다.
- 드롭다운 하단의 "새 상태" 입력 필드에서 신규 진행상태를 바로 추가할 수 있습니다.
- 설정 페이지로 이동하지 않아도 됩니다.

### 넓어진 선택 영역
- 진행상태 선택 영역의 최소 너비가 120px로 확대되어 상태 텍스트가 더 잘 보입니다.

## 2. 완료된 항목 보기 설정 유지

### 상태 저장
- "완료된 항목 보기" 스위치의 상태가 브라우저에 저장됩니다.
- 페이지를 새로고침하거나 다시 접속해도 이전에 설정한 상태가 유지됩니다.

### 사용 방법
1. 업무 기록 목록 상단의 "완료된 항목 보기" 스위치를 켜거나 끕니다.
2. 설정이 자동으로 저장됩니다.
3. 다음 접속 시에도 같은 설정이 적용됩니다.

## 3. 완료 취소 동작 개선

### 이전 문제
- 완료된 작업의 완료 취소 버튼을 누르면 종료시간이 사라지거나 현재 시간으로 변경되었습니다.

### 개선 내용
- 완료 취소 시 종료시간이 기존 값 그대로 유지됩니다.
- 상태만 COMPLETED에서 PAUSED로 변경되고, 시간 정보는 보존됩니다.

## FAQ

**Q: 주간일정에서 새 진행상태를 추가했는데 다른 화면에서도 사용할 수 있나요?**
A: 네, 진행상태는 전역으로 관리되므로 주간일정에서 추가한 상태는 모든 화면에서 사용할 수 있습니다.

**Q: 완료된 항목 보기 설정을 초기화하려면 어떻게 하나요?**
A: 브라우저의 localStorage를 초기화하거나, 스위치를 다시 토글하면 됩니다.
