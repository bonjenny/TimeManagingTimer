# TimeKeeper v0.21.4 사용설명서

## 목차
1. 시작하기
2. 간트차트 툴팁 동작
3. FAQ

## 1. 시작하기

v0.21.4는 간트차트에서 업무 막대 툴팁이 **수정** 모달을 연 뒤에도 화면에 남던 문제를 고친 패치입니다.

## 2. 간트차트 툴팁 동작

- 업무 막대에 마우스를 올리면 안내 툴팁이 표시됩니다.
- **우클릭 → 수정** 또는 **더블클릭**으로 수정 창이 열리면 툴팁은 즉시 닫히고, 수정 창이 열려 있는 동안에는 막대 위에 포인터가 있어도 툴팁이 다시 뜨지 않습니다.
- **새 업무 기록(수동)** 모달이 열려 있을 때도 동일하게 툴팁이 비활성화됩니다.

## 3. FAQ

**Q: 수정을 취소하고 막대에서 마우스를 떼었다가 다시 올리면 툴팁이 보이나요?**  
A: 네, 모달을 닫은 뒤에는 기존처럼 호버 시 툴팁이 다시 표시됩니다.
