# TimeKeeper v0.22.0 사용설명서

## 변경 요약

최근 업무 기록 및 시간관리의 '비고(Note)' 관련 기능이 대폭 개선되었습니다.

### 1. 시간관리 불러오기 기능 개선
- **이전 동작**: 일간 타이머에서 불러오기를 실행한 후 최근 업무 기록에서 비고/시간 등을 수정하고 다시 불러오기를 실행하면, 이미 불러온 항목은 내용 갱신이 되지 않았습니다.
- **개선된 동작**: 다시 불러오기를 실행할 때 시간과 비고 내용이 최신 정보로 정확하게 업데이트됩니다 (단, 사용자가 시간관리 시트에서 수동 수정한 다른 컬럼들은 유지됩니다).

### 2. 카테고리 변경 복사 동작 개선
- **이전 동작**: 복사 생성 시 새 비고를 입력하지 않으면 이전 작업의 비고가 그대로 복사되었습니다.
- **개선된 동작**: 카테고리 변경 복사 다이얼로그의 비고란이 비워진 상태로 시작하며, 새 비고를 입력하지 않으면 복사본은 비고가 없는 상태로 생성됩니다.

### 3. 세션(작업 이력) 상세 테이블 비고 컬럼 추가
- **개선된 동작**: 최근 업무 기록 리스트의 세부 세션 목록(작업 이력) 테이블에 '비고' 컬럼이 추가되었습니다. 해당 컬럼을 클릭하여 인라인 모드로 즉각적인 편집이 가능합니다.

### 4. 진행 중인 타이머의 비고 작성 지원
- **개선된 동작**: 일간 타이머 탭 상단의 진행 중인 타이머(Active Timer) 영역에, 카테고리/프로젝트 칩과 나란히 '비고'를 추가 및 수정할 수 있는 입력란(칩)이 추가되었습니다.
