## 1. 날짜 탐색 및 이동

### 개요
v0.8.0부터 날짜 선택기가 앱 최상단 중앙으로 이동하여 더욱 편리하게 날짜를 변경할 수 있습니다.

### 사용 방법
- **화면 상단 중앙**에 현재 선택된 날짜가 표시됩니다.
- `<` 버튼: 하루 전으로 이동
- `>` 버튼: 다음 날로 이동
- `오늘` 버튼 (캘린더 아이콘): 오늘 날짜로 즉시 이동
- 날짜 텍스트를 클릭하면 달력이 열려 원하는 날짜를 직접 선택할 수 있습니다 (추후 지원 예정).

## 2. 커스텀 테마 팔레트

### 개요
나만의 색상 조합을 만들어 앱의 분위기를 자유롭게 바꿀 수 있습니다.

### 설정 방법
1. **설정** 페이지로 이동합니다.
2. **테마 커스터마이징** 섹션을 찾습니다.
3. **팔레트 선택**:
   - 미리 제공되는 10가지 스타일리시한 팔레트 중 하나를 선택하거나
   - **"커스텀 팔레트 만들기"**를 선택합니다.
4. **색상 편집**:
   - 메인 색상(Primary), 강조 색상(Accent) 등을 직접 지정합니다.
   - 색상을 선택하면 앱이 자동으로 어울리는 보조 색상들을 생성합니다.
5. **저장**: 설정이 자동으로 저장되어 다음에 접속해도 유지됩니다.

## 3. 키보드 단축키

앱을 마우스 없이 더 빠르게 사용할 수 있도록 키보드 지원이 강화되었습니다.

### 전역 단축키
| 단축키 | 기능 |
|--------|------|
| **F8** | 새 작업 추가 팝업 열기 |
| **Alt + N** | 입력창 포커스 (어디서든) |
| **Alt + S** | 타이머 일시정지 / 재개 |
| **Alt + T** | 오늘 날짜로 이동 |
| **Alt + 1** | 일간 타이머 페이지로 이동 |
| **Alt + 2** | 주간 일정 페이지로 이동 |

### 모달/팝업 내 조작
| 키 | 동작 |
|----|------|
| **Enter** | 저장 / 확인 / 등록 |
| **ESC** | 취소 / 닫기 |

## 4. 기타 개선사항

### 간트 차트 사용성
- 타임라인에서 작업을 드래그하여 시간을 수정할 때, **툴팁이 손가락(마우스)을 가리지 않도록** 개선되었습니다.
- 빈 공간에 마우스를 올리면 "드래그하여 새 작업 추가" 안내가 표시됩니다.

### 점심시간 설정 (v0.7.0)
- **설정 > 업무 환경**에서 점심시간을 설정하면 총 업무 시간 계산에서 자동으로 제외됩니다.
- "점심시간 제외" 옵션을 켜고 끄면서 실제 근무 시간을 정확히 파악하세요.
