## 2. 게시글 작성

### 작성 방법

1. 건의사항 페이지에서 **"글쓰기"** 버튼을 클릭합니다.
2. 정보를 입력합니다:
   - **카테고리** (선택): 아이디어 / 버그 / 기타
   - **닉네임** (선택): 미입력 시 "익명"으로 표시
   - **제목** (필수): 건의사항 제목
   - **내용** (필수): 상세 내용
   - **비밀번호** (필수): 수정/삭제 시 필요
3. **"등록하기"** 버튼을 클릭합니다.

### 작성 팁

- **제목**은 한눈에 알아볼 수 있도록 간결하게 작성하세요.
- **내용**은 구체적으로 작성할수록 좋습니다.
  - 버그인 경우: 발생 상황, 기대 동작, 실제 동작
  - 아이디어인 경우: 필요한 이유, 예상 사용 시나리오
- **비밀번호**를 잊지 않도록 주의하세요.

## 3. 게시글 보기

### 목록 화면

게시글 목록에서 다음 정보를 확인할 수 있습니다:

- **카테고리 태그**: 아이디어(초록), 버그(빨강), 기타(회색)
- **제목**: 게시글 제목
- **작성자**: 닉네임
- **작성일시**: 등록 날짜와 시간

### 상세 화면

게시글을 클릭하면 상세 내용을 볼 수 있습니다:

- 전체 본문 내용
- 수정 여부 ("(수정됨)" 표시)
- 수정/삭제 버튼

---

## 4. 수정/삭제

### 수정하기

1. 수정하려는 게시글의 상세 화면으로 이동합니다.
2. 우측 상단의 **연필 아이콘(✏️)**을 클릭합니다.
3. **비밀번호**를 입력합니다.
4. 내용을 수정합니다.
5. **"수정하기"** 버튼을 클릭합니다.

### 삭제하기

1. 삭제하려는 게시글의 상세 화면으로 이동합니다.
2. 우측 상단의 **휴지통 아이콘(🗑️)**을 클릭합니다.
3. **비밀번호**를 입력합니다.
4. 확인 후 게시글이 삭제됩니다.

### 주의사항

- 비밀번호를 분실하면 수정/삭제가 **불가능**합니다.
- 삭제된 게시글은 **복구할 수 없습니다**.
- 비밀번호는 수정해도 **변경되지 않습니다** (기존 비밀번호 유지).

## 5. 카테고리

게시글 유형을 분류하는 카테고리입니다.

| 카테고리 | 색상 | 용도 |
|----------|------|------|
| **아이디어** | 초록 | 새로운 기능 제안, 개선 아이디어 |
| **버그** | 빨강 | 오류 신고, 비정상 동작 보고 |
| **기타** | 회색 | 질문, 의견, 기타 |

### 카테고리별 작성 가이드

**아이디어**
```
제목: [기능] 다크 모드 지원 요청
내용: 
야간에 사용할 때 눈이 피로합니다.
다크 모드 기능이 있으면 좋겠습니다.
```

**버그**
```
제목: [버그] 타이머 일시정지 후 재개 시 오류
내용:
발생 환경: Chrome 브라우저
재현 방법:
1. 타이머 시작
2. 일시정지
3. 5분 후 재개
문제: 경과 시간이 음수로 표시됨
```

## 데이터 저장

게시글은 브라우저의 LocalStorage에 저장됩니다.

- **저장 위치**: `timekeeper-feedback-posts`
- **브라우저 캐시 삭제 시** 데이터가 함께 삭제될 수 있습니다.
- 중요한 내용은 별도로 백업하세요.
