## 1. 시작하기

TimeKeeper는 업무 시간을 기록하고 관리하는 웹 애플리케이션입니다. 모든 데이터는 브라우저의 LocalStorage에 저장되므로 별도의 로그인이 필요 없습니다.

### 저장 용량 확인
화면 하단에서 현재 저장 용량을 확인할 수 있습니다:
```
현재 저장 용량: 123.4 KB (0.12 MB / 약 5MB 제한)
```

## 2. 주요 기능

### 타이머 상태
타이머(세션)는 4가지 상태를 가집니다:

| 상태 | 설명 | UI 표시 |
|------|------|---------|
| IDLE | 타이머 시작 전 | - |
| RUNNING | 작업 진행 중 | "진행 중" 칩 |
| PAUSED | 일시정지 상태 | 종료 시간(주황색) |
| COMPLETED | 작업 완료 | 종료 시간(기본색) |

### 간트차트 작업 삭제
간트차트에서 작업을 삭제하는 방법:
1. 작업 막대를 **우클릭**
2. 컨텍스트 메뉴에서 **"삭제"** 선택
3. 작업이 즉시 휴지통으로 이동되고 **토스트 알림**이 표시됨

### 휴지통 기능
삭제된 작업은 휴지통에 보관되며, 필요시 복구할 수 있습니다.

## 3. v0.12.6 변경사항

### 간트차트 삭제 UX 개선

**변경 전**
- 작업 삭제 시 "이 작업을 휴지통으로 이동하시겠습니까?" 확인 대화상자 표시
- 확인 버튼을 눌러야 삭제 진행

**변경 후**
- 확인 대화상자 없이 **즉시 삭제**
- 상단에 **초록색 토스트 알림**으로 "작업이 휴지통으로 이동되었습니다." 표시
- 3초 후 자동으로 알림이 사라짐

### 토스트 알림 색상

| 알림 종류 | 색상 | 용도 |
|-----------|------|------|
| 정보 (info) | 파란색 | 시간 충돌 자동 조정 알림 |
| 경고 (warning) | 주황색 | 유효하지 않은 시간 범위 알림 |
| 성공 (success) | 초록색 | 작업 삭제 완료 알림 |

### 삭제 후 복구
실수로 삭제한 경우 휴지통에서 작업을 복구할 수 있습니다:
1. 좌측 메뉴에서 **휴지통** 선택
2. 복구하려는 작업 찾기
3. **복구** 버튼 클릭

## 4. FAQ

### Q: 삭제 확인 없이 바로 삭제되면 실수로 지울 수 있지 않나요?
삭제된 작업은 휴지통에 보관되므로 언제든지 복구할 수 있습니다. 확인 단계를 제거하여 더 빠른 작업이 가능합니다.

### Q: 토스트 알림이 안 보여요
브라우저를 새로고침하여 v0.12.6 버전으로 업데이트하세요.

### Q: 토스트 알림을 빨리 닫을 수 있나요?
네, 알림을 클릭하거나 3초를 기다리면 자동으로 닫힙니다.

### Q: 진행 중인 작업도 삭제할 수 있나요?
네, 진행 중인 작업을 삭제하면 타이머가 자동으로 중지되고 휴지통으로 이동됩니다.
