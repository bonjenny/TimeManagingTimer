## 1. 점심시간 제외 기능

### 개요
작업 소요 시간 계산 시 점심시간을 자동으로 제외할 수 있습니다.

### 설정 방법
1. **설정** 페이지로 이동합니다.
2. **업무 환경** 섹션에서 점심시간 시작/종료를 설정합니다 (기본값: 12:00 ~ 13:00).
3. **"점심시간 소요 시간에서 제외"** 스위치를 활성화합니다.
4. **설정 저장** 버튼을 클릭합니다.

### 동작 방식
- 예: 11:30에 시작하여 13:30에 완료한 작업
  - 활성화 시: 1시간 (점심시간 1시간 제외)
  - 비활성화 시: 2시간

## 2. 휴지통 기능

### 개요
실수로 삭제한 작업을 복구할 수 있습니다.

### 사용 방법

#### 휴지통 열기
- **최근 업무 기록** 섹션 우측 상단의 🗑️ 아이콘을 클릭합니다.
- 삭제된 작업이 있으면 아이콘이 주황색으로 표시됩니다.

#### 복원하기
1. 휴지통 모달에서 복원할 작업을 찾습니다.
2. ↩️ (복원) 버튼을 클릭합니다.
3. 작업이 기존 기록 목록으로 복원됩니다.

#### 영구 삭제
1. 🗑️ (삭제) 버튼을 클릭합니다.
2. 확인 메시지에서 "확인"을 클릭합니다.
3. 해당 작업이 완전히 삭제됩니다.

#### 휴지통 비우기
- **휴지통 비우기** 버튼으로 모든 삭제된 작업을 일괄 삭제할 수 있습니다.

## 3. 주간 일정 이동

### 개요
주간 일정 페이지에서 이전/다음 주를 탐색할 수 있습니다.

### 사용 방법
1. **주간 일정** 페이지로 이동합니다.
2. 필터에서 **주간**을 선택합니다.
3. `<` 버튼: 이전 주로 이동
4. `>` 버튼: 다음 주로 이동
5. 📅 버튼: 이번 주로 돌아가기 (이번 주가 아닌 경우 표시)

### 표시 형식
- 이번 주: `이번 주 (1/20 ~ 1/26)`
- 지난 주: `지난 주 (1/13 ~ 1/19)`
- 그 외: `1/6 ~ 1/12`

## 4. 복사 템플릿

### 개요
주간 일정 데이터를 클립보드에 복사하여 보고서 작성에 활용할 수 있습니다.

### 사용 방법
1. **주간 일정** 페이지에서 원하는 기간을 선택합니다.
2. **복사** 버튼을 클릭합니다.
3. 복사 형식을 선택합니다:

#### 상세형
날짜, 작업명, 시간, 소요시간, 카테고리를 포함한 상세 목록입니다.

```
날짜 | 작업명 | 시간 | 소요시간 | 카테고리
------------------------------------------------------------
2026-01-26 | API 개발 | 09:30~11:30 | 120분 | 개발
2026-01-26 | 주간 회의 | 14:00~15:30 | 90분 | 회의
```

#### 요약형
게시판 번호별로 그룹화된 요약 정보입니다.

```
[#12345] API 개발, DB 설계 - 3시간 30분 (4건)
[#12346] 주간 회의 - 1시간 30분 (1건)

총 업무시간: 5시간 0분
```

## 5. 단축키 안내

| 단축키 | 기능 |
|--------|------|
| **F8** | 새 작업 추가 팝업 열기 |
| **Enter** | 타이머 시작 (입력창 포커스 시) |
| **Alt + N** | 입력창 포커스 |
| **Alt + S** | 타이머 일시정지/재개 |
| **Alt + 1** | 일간 타이머 페이지 이동 |
| **Alt + 2** | 주간 일정 페이지 이동 |
| **Alt + T** | 오늘로 이동 |

단축키는 **설정 > 업무 환경** 섹션에서도 확인할 수 있습니다.

## FAQ

### Q: 점심시간 제외가 적용되지 않아요.
A: 설정 페이지에서 "점심시간 소요 시간에서 제외" 스위치가 활성화되어 있는지 확인하고, **설정 저장** 버튼을 눌렀는지 확인해주세요.

### Q: 휴지통에서 복원한 작업이 안 보여요.
A: 복원된 작업은 원래 기록된 날짜에 표시됩니다. 일간 타임라인에서 해당 날짜로 이동하거나, 주간 일정에서 해당 기간을 선택해주세요.

### Q: 복사한 템플릿이 엑셀에서 깨져 보여요.
A: 엑셀보다는 텍스트 에디터나 이메일 본문에 붙여넣기 하시는 것을 권장합니다. CSV 다운로드 기능을 사용하면 엑셀 호환 파일을 받을 수 있습니다.
