## 1. 시작하기

TimeKeeper는 업무 시간을 기록하고 관리하는 도구입니다. v0.13.0에서는 미완료 업무 표시 기능이 크게 개선되었습니다.

## 2. 주요 기능

### 타이머 기능
- 업무 시작/정지/완료
- 프로젝트 코드 및 카테고리 지정
- 간트 차트를 통한 시각적 표시

### 업무 기록
- 일자별 업무 기록 조회
- 업무 그룹화 및 세션 관리
- 휴지통 기능

## 3. 새로운 기능: 미완료 업무 표시 개선

### 3.1 미완료 업무 자동 표시

**이전 버전**: 업무는 시작한 날짜에만 표시되었습니다.

**v0.13.0**: PAUSED(일시정지) 또는 RUNNING(진행 중) 상태의 업무는 시작 날짜와 무관하게 **오늘 날짜에 항상 표시**됩니다.

예시:
- 1월 27일에 시작한 업무가 아직 완료되지 않음 (PAUSED 상태)
- 1월 28일 오늘 날짜를 선택해도 해당 업무가 목록에 표시됨

### 3.2 이전 날짜 시작 표시

미완료 업무가 이전 날짜에 시작된 경우, 시작 시간에 날짜가 함께 표시됩니다.

형식: `M/D HH:mm ~`

예시:
- 1/27 06:39 ~ 일시정지
- 1/26 14:30 ~ 진행 중

### 3.3 총시간/오늘시간 컬럼 분리

업무 목록에서 시간 정보가 두 개의 컬럼으로 분리되었습니다:

| 컬럼 | 설명 |
|------|------|
| **총시간** | 해당 업무의 전체 작업 시간 (모든 날짜 포함) |
| **오늘시간** | 오늘 날짜 범위(06:00 ~ 익일 06:00) 내 작업 시간만 |

이를 통해:
- 업무의 전체 진행 시간을 파악할 수 있습니다
- 오늘 해당 업무에 투자한 시간을 별도로 확인할 수 있습니다

## 4. FAQ

### Q: 미완료 업무가 오늘 목록에 보이지 않아요
A: 업무 상태가 COMPLETED(완료)인지 확인하세요. 완료된 업무는 해당 날짜에만 표시됩니다.

### Q: 총시간과 오늘시간이 다른 이유는?
A: 해당 업무가 여러 날에 걸쳐 진행된 경우입니다. 총시간은 모든 세션의 합계이고, 오늘시간은 오늘 작업한 시간만 표시합니다.

### Q: 이전 날짜를 선택하면 미완료 업무가 안 보여요
A: 미완료 업무는 "오늘" 날짜를 선택했을 때만 자동으로 표시됩니다. 이전 날짜에서는 해당 날짜에 시작된 세션만 표시됩니다.
