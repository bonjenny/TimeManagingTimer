## 1. 시작하기

TimeKeeper는 업무 시간을 기록하고 관리하는 도구입니다. v0.13.4에서는 화면 크기(줌) 설정 기능이 추가되었습니다.

## 2. 주요 기능

### 타이머 기능
- 업무 시작/정지/완료
- 프로젝트 코드 및 카테고리 지정
- 간트 차트를 통한 시각적 표시

### 업무 기록
- 일자별 업무 기록 조회
- 업무 그룹화 및 세션 관리
- 휴지통 기능

## 3. 새로운 기능: 화면 크기 설정

### 3.1 화면 크기 설정이란?

화면의 전체적인 크기를 조절하여 사용자의 시력이나 모니터 크기에 맞게 앱을 사용할 수 있습니다.

### 3.2 사용 방법

1. **설정 페이지 이동**: 상단 메뉴에서 "설정" 탭을 클릭합니다.
2. **화면 크기 섹션 찾기**: 페이지 상단에 "화면 크기" 섹션이 있습니다.
3. **크기 선택**: 5가지 옵션 중 원하는 크기를 클릭합니다.

### 3.3 화면 크기 옵션

| 옵션 | 설명 |
|------|------|
| **작게 (80%)** | 더 많은 콘텐츠를 화면에 표시 |
| **약간 작게 (90%)** | 기본보다 조금 작은 크기 |
| **보통 (100%)** | 기본 크기 (권장) |
| **약간 크게 (110%)** | 기본보다 조금 큰 크기 |
| **크게 (120%)** | 더 큰 글씨와 UI 요소 |

### 3.4 특징

- **자동 저장**: 화면 크기를 변경하면 즉시 localStorage에 저장됩니다.
- **새로고침 후 유지**: 브라우저를 새로고침하거나 다시 열어도 설정이 유지됩니다.
- **즉시 적용**: 옵션을 클릭하면 바로 화면 크기가 변경됩니다.
- **기본값 복원**: 설정 페이지 하단의 "기본값 복원" 버튼으로 100%로 초기화할 수 있습니다.

## 4. FAQ

### Q: 화면 크기 설정이 저장되지 않아요
A: 브라우저의 localStorage가 활성화되어 있는지 확인하세요. 시크릿 모드에서는 데이터가 저장되지 않을 수 있습니다.

### Q: 화면 크기 변경 후 레이아웃이 이상해요
A: 특정 화면 크기에서 일부 UI가 잘리거나 겹칠 수 있습니다. "보통 (100%)" 옵션을 사용하시거나, 브라우저 창 크기를 조절해 보세요.

### Q: 기본값으로 돌아가려면?
A: 설정 페이지 하단의 "기본값 복원" 버튼을 클릭하면 화면 크기가 100%로 초기화됩니다.

### Q: 다른 기기에서도 같은 설정을 사용하려면?
A: 현재 설정은 각 기기의 브라우저에 개별 저장됩니다. 다른 기기에서 같은 설정을 사용하려면 해당 기기에서 다시 설정하거나, 데이터 내보내기/가져오기 기능을 사용하세요.
