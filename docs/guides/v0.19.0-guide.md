## 주요 기능

### 시간관리 탭

시간관리 탭은 일간 타이머의 작업 기록을 표 형식으로 정리하고 관리할 수 있는 기능입니다.

#### 기본 사용법

1. **데이터 자동 로드**:
   - 시간관리 탭에 접근하면 해당 날짜의 데이터가 자동으로 로드됩니다
   - 같은 작업명의 로그는 자동으로 합쳐져 시간이 합산됩니다

2. **날짜 변경**:
   - 이전/다음 화살표: 하루씩 이동
   - 오늘 버튼: 현재 날짜로 빠르게 이동
   - 달력 아이콘: 특정 날짜 선택

3. **데이터 편집**:
   - 셀을 클릭하여 직접 수정
   - Enter 키 또는 다른 곳 클릭 시 자동 저장
   - 드롭다운 제공: 작업(프로젝트), 업무형, 카테고리

## 시간관리 탭 상세 가이드

### 1. 표 컬럼 설명

| 컬럼 | 설명 | 편집 방법 |
|------|------|----------|
| 체크박스 | 삭제할 행 선택 | 클릭 |
| 작업 | 프로젝트명 | 드롭다운 선택 |
| 업무형 | 작업 또는 개발 | 드롭다운 선택 |
| 거래형(일정명) | 작업 제목 | 텍스트 입력 |
| 카테고리 코드 | 숫자 코드 (예: 05) | 드롭다운 선택 |
| 카테고리명 | 카테고리 이름 | 드롭다운 선택 |
| 시간(분) | 작업 소요 시간 | 숫자 입력 |
| 비고 | 메모 및 특이사항 | 텍스트 입력 |

### 2. 업무형 자동 분류 시스템

시간관리 탭에서 데이터를 불러올 때, 업무형은 다음 우선순위로 자동 결정됩니다:

```
1순위: 프리셋 작업
   ↓ (해당 없음)
2순위: 프로젝트별 설정
   ↓ (해당 없음)
3순위: 카테고리별 설정
   ↓ (해당 없음)
4순위: 기본 업무형
```

#### 예시

**상황**: A25_05591 프로젝트에서 "회의" 카테고리로 작업을 했습니다.

- 프리셋 작업? ❌ → 다음 단계
- 프로젝트별 설정? ✅ A25_05591 = "작업" → **결과: "작업"**

**상황**: 프로젝트 없이 "개발" 카테고리로 작업을 했습니다.

- 프리셋 작업? ❌ → 다음 단계
- 프로젝트별 설정? ❌ (프로젝트 없음) → 다음 단계
- 카테고리별 설정? ❌ ("개발"은 미설정) → 다음 단계
- 기본 업무형? ✅ "작업" (기본값) → **결과: "작업"**

### 3. 업무형 설정 관리

#### 3-1. 기본 업무형 설정

모든 작업의 기본 업무형을 지정합니다.

**설정 위치**: 시간관리 탭 상단 드롭다운

**사용 예**:
- 주로 개발 작업을 한다면 "개발"로 설정
- 주로 일반 업무를 한다면 "작업"으로 설정

#### 3-2. 카테고리별 업무형 설정

특정 카테고리에 대한 기본 업무형을 지정합니다.

**설정 방법**:
1. 시간관리 탭 상단의 회색 톱니바퀴 아이콘 클릭
2. "카테고리별 업무형 설정" 다이얼로그 열림
3. 카테고리 선택 (Autocomplete)
4. 업무형 선택 (작업/개발)
5. "추가" 버튼 클릭

**기본 설정**:
- 테스트오류수정: "작업"
- 센터오류수정: "작업"
- 질의응답: "작업"

**삭제 방법**:
- 설정된 카테고리 옆의 휴지통 아이콘 클릭

#### 3-3. 프로젝트별 업무형 설정 ⭐ NEW

특정 프로젝트 코드에 대한 기본 업무형을 지정합니다.

**설정 방법**:
1. 시간관리 탭 상단의 **파란색** 톱니바퀴 아이콘 클릭
2. "프로젝트별 업무형 설정" 다이얼로그 열림
3. 프로젝트 선택 (Autocomplete, 프로젝트명 + 코드로 표시)
4. 업무형 선택 (작업/개발)
5. "추가" 버튼 클릭

**기본 설정**:
- A25_05591: "작업"

**사용 예시**:
- A26_12345 프로젝트는 항상 개발 작업 → "개발"로 설정
- B26_00001 프로젝트는 항상 일반 업무 → "작업"으로 설정

**삭제 방법**:
- 설정된 프로젝트 옆의 휴지통 아이콘 클릭

### 4. 엑셀 Export ⭐ NEW

시간관리 데이터를 엑셀 파일로 내보낼 수 있습니다.

**사용 방법**:
1. 시간관리 탭에서 원하는 날짜 선택
2. 상단의 초록색 "엑셀 Export" 버튼 클릭
3. 파일이 자동으로 다운로드됨

**파일명 형식**: `시간관리_YYYY-MM-DD.xlsx`
- 예: `시간관리_2026-03-13.xlsx`

**Export 내용**:
- 프로젝트 코드 → 프로젝트명으로 변환
- 모든 컬럼 데이터 포함
- 마지막 행에 시간 합계 자동 추가

**활용 방법**:
- 일일 업무 보고서 작성
- 프로젝트별 시간 집계
- 월별 업무 통계 작성

### 5. 행 추가/삭제

#### 새 행 추가
1. 상단의 "새 행 추가" 버튼 클릭
2. 빈 행이 추가됨
3. 각 셀을 클릭하여 데이터 입력

#### 행 삭제
1. 삭제할 행의 체크박스 선택
2. 상단의 "삭제 (N)" 버튼 클릭
3. 선택된 행이 삭제됨

**Tip**: 헤더의 체크박스를 클릭하면 전체 행을 한 번에 선택/해제할 수 있습니다.

## FAQ

### Q1. 시간관리 데이터를 수정하면 일간 타이머도 변경되나요?

**A**: 아니요. 시간관리 탭의 데이터는 일간 타이머와 별도로 관리됩니다. 시간관리에서 수정해도 원본 타이머 로그는 변경되지 않습니다.

### Q2. 같은 데이터를 여러 번 불러오면 중복되나요?

**A**: 아니요. 시스템은 원본 로그 ID를 추적하여 이미 불러온 로그는 다시 추가하지 않습니다.

### Q3. 업무형이 자동으로 잘못 설정됩니다. 어떻게 하나요?

**A**: 다음 방법으로 해결할 수 있습니다:
1. 해당 프로젝트나 카테고리에 대한 업무형 설정 추가
2. 기본 업무형 변경
3. 불러온 후 수동으로 수정

### Q4. 엑셀 Export 버튼이 비활성화되어 있습니다.

**A**: 현재 날짜에 데이터가 없는 경우 버튼이 비활성화됩니다. "일간 타이머에서 불러오기" 버튼을 클릭하거나 "새 행 추가"로 데이터를 추가하세요.

### Q5. 프로젝트별 설정과 카테고리별 설정이 충돌하면 어떻게 되나요?

**A**: 프로젝트별 설정이 더 높은 우선순위를 가지므로, 프로젝트별 설정이 적용됩니다.

예시:
- 프로젝트 A25_05591: "작업"
- 카테고리 "개발": "개발"
- A25_05591 프로젝트에서 "개발" 카테고리로 작업 → 결과: "작업" (프로젝트 설정 우선)

### Q6. 엑셀 파일의 프로젝트명이 "-"로 표시됩니다.

**A**: 해당 작업에 프로젝트가 지정되지 않았거나, 프로젝트 코드가 시스템에 등록되지 않은 경우입니다. 설정 페이지에서 프로젝트를 추가하거나, 시간관리 탭에서 올바른 프로젝트를 선택하세요.

### Q7. 시간 합계를 확인하고 싶습니다.

**A**: 두 가지 방법이 있습니다:
1. 시간관리 탭 하단에 "총 N건" 옆에 표시됨
2. 엑셀 Export 시 마지막 행에 합계가 자동으로 추가됨
