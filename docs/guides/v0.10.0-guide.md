# TimeKeeper v0.10.0 사용자 가이드

TimeKeeper v0.10.0 버전의 새로운 기능과 변경된 사용 방법을 안내합니다.

## 1. 간트 차트 (Timeline) 활용하기

일간 타이머 화면의 중앙에 위치한 간트 차트(Timeline)의 사용성이 대폭 개선되었습니다.

### 작업 수정하기
- **방법 1 (더블 클릭)**: 수정하고 싶은 작업 막대를 **더블 클릭**하세요. 수정 팝업이 바로 열립니다.
- **방법 2 (우클릭 메뉴)**: 작업 막대를 **마우스 오른쪽 버튼**으로 클릭하고 **[수정]** 메뉴를 선택하세요.

### 작업 삭제하기
- 작업 막대를 **마우스 오른쪽 버튼**으로 클릭하고 **[삭제]** 메뉴를 선택하세요.
- 확인 창에서 '확인'을 누르면 해당 작업이 휴지통으로 이동합니다.

### 작업 시간 조절하기
- (기존 기능) 완료된 작업의 경우, 작업 막대의 양쪽 끝부분에 마우스를 올리면 커서가 변경됩니다. 드래그하여 시작/종료 시간을 조절할 수 있습니다.

---

## 2. 테마 설정 변경하기

### 다크 모드 전환
- 화면 오른쪽 상단(앱 바)의 **해/달 아이콘**을 클릭하여 라이트 모드와 다크 모드를 전환할 수 있습니다.

### 앱 컬러 변경 (작업 팔레트)
- **[설정] > [테마 및 컬러 팔레트]** 메뉴로 이동합니다.
- 원하는 '작업 컬러 팔레트'(예: 네이비 & 오렌지, 오션 블루 등)를 선택하세요.
- 팔레트를 변경하면 **간트 차트의 작업 색상**뿐만 아니라 **앱 전체의 주요 색상(버튼, 상단 바 등)**도 함께 변경됩니다.
- 나만의 색상을 원한다면 '사용자 커스텀 팔레트'를 만들어보세요.

## 3. 자주 묻는 질문 (FAQ)

**Q. "빈 영역을 드래그하여..." 툴팁이 안 보여요.**
A. v0.10.0부터는 시각적 편의를 위해 해당 툴팁을 제거했습니다. 여전히 빈 영역을 드래그하여 새 작업을 추가할 수 있습니다.

**Q. 테마 커스터마이징 메뉴가 사라졌어요.**
A. 복잡한 색상 설정을 줄이고 '작업 컬러 팔레트' 선택 하나로 앱 전체 분위기를 바꿀 수 있도록 통합되었습니다.
