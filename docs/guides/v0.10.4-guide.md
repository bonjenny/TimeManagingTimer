## 1. 새로운 기능

v0.10.4에서는 사용자 편의성을 크게 개선했습니다:

- **카테고리 인라인 관리**: 드롭다운에서 바로 카테고리 추가/삭제
- **진행상태 커스터마이징**: 나만의 진행상태 추가 가능
- **인라인 편집**: 클릭 한 번으로 기록명, 프로젝트 코드 수정
- **자동완성 관리**: 불필요한 자동완성 항목 삭제

---

## 2. 카테고리 인라인 관리

### 사용 방법

1. **카테고리 드롭다운 클릭**
2. **기존 카테고리 선택** 또는
3. **하단의 "새 카테고리" 입력란에 입력 후 추가 버튼 클릭**
4. **각 카테고리 옆 X 버튼으로 삭제**

### 적용 위치
- 타이머 입력 영역
- 기록 수정 모달
- 간트 차트 작업 생성/수정
- 프리셋 추가/수정
- 설정 페이지

---

## 3. 진행상태 커스터마이징

### 기본 진행상태
- 완료
- 진행중
- 대기
- 보류

### 새 진행상태 추가 방법

**방법 1: 주간일정에서 직접 추가**
1. 주간일정 페이지로 이동
2. 프로젝트의 진행상태 드롭다운 클릭
3. 하단 입력란에 새 상태 입력
4. Enter 또는 + 버튼 클릭

**방법 2: 설정에서 관리**
1. 설정 페이지로 이동
2. "진행상태 관리" 섹션 찾기
3. 표시명 입력 후 추가 버튼 클릭
4. 필요 시 "기본값으로 초기화" 클릭

## 4. 인라인 편집 기능

### 기록명 편집
1. **기록 목록에서 기록명 클릭**
2. **텍스트 필드가 나타남**
3. **수정 후 Enter 또는 포커스 아웃**

### 프로젝트 코드 편집
1. **프로젝트 코드 Chip 클릭**
2. **Autocomplete 드롭다운 표시**
3. **기존 프로젝트 선택 또는 직접 입력**

### 상세 수정 (모달)
- 기록 행 우측의 **연필 아이콘** 클릭
- 시작/종료 시간, 카테고리 등 상세 정보 수정

## 5. 자동완성 항목 삭제

### 기록명 자동완성 삭제
1. 기록명 입력란 클릭하여 자동완성 목록 표시
2. 삭제할 항목 옆 X 버튼 클릭
3. 해당 항목은 더 이상 자동완성에 표시되지 않음
4. **참고**: 실제 기록 데이터는 삭제되지 않음

### 프로젝트 자동완성 삭제
1. 프로젝트 코드/명 입력란 클릭
2. 삭제할 프로젝트 옆 X 버튼 클릭
3. **주의**: 프로젝트가 저장 목록에서 영구 삭제됨

---

## 6. 기타 개선사항

### UI 개선
- **간트 차트 프로젝트명**: 긴 이름은 말줄임표(...) 처리
- **버튼 순서 변경**: 완료 버튼이 가장 왼쪽에 배치
- **Footer 안내 문구**: 데이터 저장 방식 안내 추가

### 데이터 저장 안내
> 모든 기록은 내 컴퓨터(브라우저)에 저장되며, 서버로 전송되지 않습니다.
> 저장 공간이 부족할 경우 오래된 데이터부터 자동 삭제될 수 있습니다.

## FAQ

### Q: 삭제한 카테고리를 복구할 수 있나요?
A: 설정 페이지에서 "기본값으로 초기화" 버튼을 클릭하면 기본 카테고리로 복구됩니다.

### Q: 자동완성에서 제외한 기록명을 다시 표시할 수 있나요?
A: 현재 버전에서는 불가능합니다. 같은 제목으로 새 기록을 생성하면 다시 표시됩니다.

### Q: 프로젝트를 삭제하면 관련 기록도 삭제되나요?
A: 아니요. 프로젝트 목록에서만 삭제되며, 기존 기록의 프로젝트 코드는 유지됩니다.
