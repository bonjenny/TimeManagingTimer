## 1. 업데이트 요약

v0.10.4에서는 **건의사항 게시판**이 공용 게시판으로 업그레이드되었습니다.

### 주요 변경사항

| 변경 전 (v0.10.3 이하) | 변경 후 (v0.10.4) |
|---|---|
| 내 브라우저에서만 보임 | 모든 사용자가 공유 |
| 브라우저 캐시 삭제 시 데이터 손실 | 클라우드에 영구 저장 |
| 오프라인에서도 사용 가능 | 인터넷 연결 필요 |

## 2. 건의사항 게시판 사용법

### 2.1 게시판 접속
1. 상단 네비게이션 바에서 **건의사항** 클릭
2. 게시글 목록이 표시됩니다

### 2.2 글쓰기
1. 우측 상단 **글쓰기** 버튼 클릭
2. 카테고리 선택 (아이디어 / 버그 / 기타)
3. 닉네임 입력 (선택, 미입력 시 "익명")
4. 제목 입력 (필수)
5. 내용 입력 (필수)
6. 비밀번호 입력 (필수, 수정/삭제 시 필요)
7. **등록하기** 클릭

### 2.3 게시글 보기
1. 목록에서 게시글 클릭
2. 본문 및 댓글 확인
3. 좌측 상단 **목록으로** 버튼으로 돌아가기

### 2.4 수정 및 삭제
1. 상세 화면에서 **수정(✏️)** 또는 **삭제(🗑️)** 아이콘 클릭
2. 작성 시 입력한 비밀번호 입력
3. 확인 버튼 클릭

### 2.5 댓글 달기
1. 상세 화면 하단 댓글 영역으로 이동
2. 닉네임 입력 (선택)
3. 비밀번호 입력 (필수, 삭제 시 필요)
4. 댓글 내용 입력
5. **댓글 등록** 클릭

### 2.6 새로고침
- 목록 화면에서 새로고침 버튼(🔄)을 클릭하면 최신 게시글을 불러옵니다
- 다른 사용자가 새로 작성한 글을 확인할 수 있습니다

## 3. 자주 묻는 질문

### Q1. 이전에 작성한 게시글이 사라졌어요
v0.10.4 이전 버전에서 작성한 게시글은 브라우저 LocalStorage에 저장되어 있었습니다. 새 버전은 Firebase 클라우드를 사용하므로, 기존 게시글은 자동으로 이전되지 않습니다. 필요한 내용은 다시 작성해주세요.

### Q2. 게시글이 안 보여요 / 로딩이 안 돼요
1. 인터넷 연결 상태를 확인해주세요
2. 페이지를 새로고침(F5) 해보세요
3. 새로고침 버튼(🔄)을 클릭해보세요

### Q3. 비밀번호를 잊어버렸어요
비밀번호는 복구할 수 없습니다. 해당 게시글/댓글의 수정 및 삭제가 불가능합니다.

### Q4. 게시글을 수정하면 비밀번호도 바뀌나요?
아니요, 비밀번호는 최초 작성 시 설정한 값이 유지됩니다.

### Q5. 다른 사람의 글이 보여요
정상입니다! v0.10.4부터 건의사항 게시판은 모든 사용자가 공유하는 공용 게시판입니다. 다른 사용자의 아이디어나 버그 리포트를 확인하고, 댓글로 소통할 수 있습니다.

### Q6. 부적절한 게시글을 발견했어요
현재 버전에서는 관리자 기능이 없습니다. 추후 업데이트에서 신고 기능을 추가할 예정입니다.
