## 1. 시작하기

### 기본 사용법
1. 상단 입력창에 작업 이름 입력
2. Enter 또는 시작 버튼 클릭
3. 타이머가 자동으로 시작됨

### 동일 작업 재시작 (v0.12.1 수정)
- 이미 완료된 동일 이름의 작업이 있어도 기존 세션은 그대로 유지됨
- 새 세션만 새로 시작되며, 기존 완료된 세션의 종료시간은 보존됨

## 2. 타이머 세션 관리

### 세션 상태
| 상태 | 설명 | 종료시간 |
|---|---|---|
| RUNNING | 진행 중 | 없음 |
| PAUSED | 일시정지됨 | 일시정지 시점 |
| COMPLETED | 완료됨 | 완료 시점 |

### 최근 업무 기록 버튼
- **▶ (재생 아이콘)**: 해당 작업으로 새 타이머 시작
- **⏸ (일시정지 아이콘)**: 진행 중인 작업 → 일시정지 후 기록으로 이동
- **✓ (체크 아이콘)**: 완료 상태 토글

### 세션 시간 편집
1. 세션 목록 펼치기 (작업 행 클릭)
2. 시작/종료 시간 클릭
3. 시간 위젯으로 수정
4. Enter 또는 바깥 클릭으로 저장

## 3. 간트차트 시간 조정

### 드래그 핸들
| 핸들 위치 | 기능 |
|---|---|
| 왼쪽 | 시작 시간 조정 |
| 오른쪽 | 종료 시간 조정 (COMPLETED/PAUSED) |

### 짧은 작업 (20분 이내)
- 핸들이 겹쳐있어 구분이 어려움
- 오른쪽 핸들 클릭 후 **왼쪽으로** 드래그 → 시작시간 조정으로 자동 전환

### 우클릭 메뉴
- 수정: 제목, 프로젝트 코드, 카테고리 변경
- 삭제: 휴지통으로 이동

## 4. 스마트 스냅 기능

### 스냅 임계값
| 대상 | 임계값 |
|---|---|
| 기존 로그 시작/종료 | 15분 |
| 업무 시작 (09:00) | 5분 |
| 점심 시작 (12:00) | 5분 |
| 점심 종료 (13:00) | 5분 |
| 업무 종료 (18:00) | 5분 |

### 점심시간 자동 회피
- 세션이 점심시간(12:00~13:00)과 겹치면 자동 조정
- 세션 시작이 점심 내부 → 점심 종료로 밀기
- 세션 종료가 점심 내부 → 점심 시작으로 당기기

## 5. 카테고리/상태 관리

### 새 카테고리 추가 (v0.12.1 개선)
1. 카테고리 드롭다운 클릭
2. 하단의 "새 카테고리" 입력 필드에 이름 입력
3. Enter 키 또는 + 버튼 클릭
4. 드롭다운이 유지되어 여러 카테고리 연속 추가 가능

### 새 상태 추가 (v0.12.1 개선)
1. 상태 드롭다운 클릭
2. 하단의 "새 상태" 입력 필드에 이름 입력
3. Enter 키 또는 + 버튼 클릭
4. 드롭다운이 유지되어 여러 상태 연속 추가 가능

### 카테고리/상태 삭제
- 각 항목 옆의 X 버튼 클릭

## 6. FAQ

### Q: 일시정지한 작업의 종료시간이 왜 표시되나요?
A: v0.12.0부터 일시정지 시 해당 세션이 종료된 것으로 처리됩니다. 이후 재시작하면 새 세션이 생성됩니다.

### Q: 09:00 근처로 드래그했는데 정확히 09:00에 맞춰지네요?
A: 5분 이내 거리에서 자동 스냅됩니다. 정밀 조정이 필요하면 세션 목록에서 직접 시간을 입력하세요.

### Q: 점심시간에 작업을 기록하고 싶어요
A: 설정 페이지에서 "점심시간 소요 시간에서 제외" 옵션을 끄면 점심시간 회피가 비활성화됩니다.

### Q: 다크모드에서 시간 위젯이 안 보여요
A: v0.12.0에서 수정되었습니다. 시간 텍스트와 아이콘이 흰색으로 표시됩니다.

### Q: 카테고리 추가 시 드롭다운이 닫혀버려요 (v0.12.1 해결)
A: v0.12.1에서 수정되었습니다. 이제 "새 카테고리" 입력 필드를 클릭하거나 입력해도 드롭다운이 유지됩니다.

### Q: 모달 하단에서 드롭다운이 잘려요 (v0.12.1 해결)
A: v0.12.1에서 수정되었습니다. 공간이 부족하면 드롭다운이 자동으로 위쪽으로 열립니다.

### Q: 완료된 작업을 재시작했는데 이전 세션 종료시간이 사라졌어요 (v0.12.1 해결)
A: v0.12.1에서 수정되었습니다. 이제 완료된 작업을 재시작해도 기존 세션의 종료시간이 유지됩니다.

### Q: 프리셋 패널에서 제목이 버튼과 겹쳐요 (v0.12.1 해결)
A: v0.12.1에서 수정되었습니다. 긴 제목은 자동으로 `...`으로 축약되며, 마우스를 올리면 전체 텍스트를 확인할 수 있습니다.

### Q: 카테고리/상태 드롭다운이 다른 곳을 클릭해도 안 닫혀요 (v0.12.1 해결)
A: v0.12.1에서 수정되었습니다. 이제 드롭다운 외부를 클릭하면 정상적으로 닫힙니다.
