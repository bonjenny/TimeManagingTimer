# TimeKeeper v0.14.3 사용설명서

v0.14.1 사용설명서를 기준으로 하며, v0.14.3에서 변경·추가된 주간 일정 관련 내용만 정리합니다.

## v0.14.3 변경 사항

### 주간 일정 > 복사 미리보기

- **복사 형식**: 간단형(텍스트) / 상세형(HTML, 잡 라벨 무색) / 라벨형(HTML, 잡 라벨 색상) 중 선택 후 "복사"로 클립보드 복사.
- **잡 색상 설정**: **라벨형**을 선택했을 때만 미리보기 하단에 "잡 색상 설정" 버튼이 보입니다. 버튼을 누르면 해당 주에 등장한 잡 + localStorage에 있는 모든 프로젝트(배포 캘린더에 없는 잡 포함)에 대해 색상을 설정할 수 있으며, 배포 캘린더와 동일 스토어를 사용해 색상이 동기화됩니다.
- **상세형**: 잡 라벨([코드] 이름)에는 배경색이 적용되지 않고, 진행상태·시작일자·누적시간 등 메타 정보만 표시됩니다.
- **간단형**: 잡 라인 아래 업무목록(> 작업명 (누적시간: ...))에 들여쓰기(공백 2칸)가 적용되어 구분이 쉽습니다.

## 단축키 (요약)

| 단축키 | 기능 |
|--------|------|
| `Alt + 3` | 배포 캘린더 |
| `Alt + 1` | 일간 타이머 |
| `Alt + 2` | 주간 일정 |
| `Alt + T` | 오늘로 이동 |
| `F8` | 새 작업 추가 |

전체 기능은 [v0.14.1 사용설명서](v0.14.1-guide.md)를 참고하세요.
