# TimeKeeper v0.21.5 사용설명서

## 변경 요약

간트차트 작업 막대 **툴팁** 동작이 개선되었습니다.

### 이전 동작 (v0.21.4)
- 우클릭 메뉴를 열었다가 바깥을 클릭하면 → 툴팁이 사라지지 않음
- 또는 사라져도 → 다시 호버해도 툴팁이 안 나옴

### 개선된 동작 (v0.21.5)
- 우클릭 메뉴가 열려 있으면 → 툴팁 즉시 숨김
- 수정/생성 모달이 열려 있으면 → 툴팁 즉시 숨김
- 메뉴·모달을 닫은 뒤 → 같은 막대 위에 커서가 있으면 자연스럽게 툴팁 다시 표시
- 커서를 빼고 다시 호버해도 정상 동작
