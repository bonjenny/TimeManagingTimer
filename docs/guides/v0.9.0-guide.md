## 1. 건의사항 게시판

### 개요
앱 사용 중 불편한 점이나 새로운 아이디어가 있다면 건의사항 게시판을 이용해보세요. 다른 사용자의 의견을 보고 댓글로 소통할 수도 있습니다. (현재 버전에서는 데이터가 사용자 개인 브라우저에만 저장됩니다)

### 접속 방법
- 화면 상단 메뉴에서 **[건의사항]** 탭을 클릭하세요.

## 2. 게시글 작성하기

1. 게시판 화면 우측 상단의 **[글쓰기]** 버튼을 클릭합니다.
2. **카테고리**를 선택합니다 (아이디어 / 버그 / 기타).
3. **닉네임**을 입력합니다 (입력하지 않으면 '익명'으로 표시됨).
4. **제목**과 **내용**을 상세히 작성합니다.
5. **비밀번호**를 설정합니다.
   - **주의**: 이 비밀번호는 나중에 글을 수정하거나 삭제할 때 꼭 필요하므로 잊어버리지 않도록 주의하세요!
6. **[등록하기]** 버튼을 누르면 작성이 완료됩니다.

---

## 3. 댓글 남기기

1. 목록에서 원하는 게시글을 클릭합니다.
2. 상세 화면 하단의 댓글 입력창에 내용을 적습니다.
3. 닉네임과 **댓글 비밀번호**를 입력합니다.
   - 댓글 삭제 시 이 비밀번호가 필요합니다.
4. **[댓글 등록]** 버튼을 누릅니다.

---

## 4. 수정 및 삭제

### 게시글 수정/삭제
1. 게시글 상세 화면 상단의 ✏️(수정) 또는 🗑️(삭제) 아이콘을 누릅니다.
2. 글 작성 시 설정했던 **비밀번호**를 입력합니다.
3. 비밀번호가 일치하면 수정 화면으로 이동하거나 글이 삭제됩니다.

### 댓글 삭제
1. 자신이 쓴 댓글 우측의 🗑️ 아이콘을 누릅니다.
2. 댓글 작성 시 설정했던 **비밀번호**를 입력합니다.
3. 확인을 누르면 댓글이 삭제됩니다.

## FAQ

### Q: 제가 쓴 글이 다른 사람에게도 보이나요?
A: 아니요. 현재 버전(v0.9.0)은 **로컬 모드**로 동작합니다. 작성하신 글은 **사용자 본인의 브라우저에만 저장**되며, 개발자나 다른 사용자에게 전송되지 않습니다. 개인 메모장이나 아이디어 정리용으로 활용하세요.

### Q: 비밀번호를 잊어버렸어요.
A: 보안을 위해 비밀번호 찾기 기능은 제공되지 않습니다. 로컬 저장소 데이터를 초기화(설정 > 데이터 관리)하면 모든 글이 삭제되지만, 개별 글의 비밀번호를 복구할 수는 없습니다.
