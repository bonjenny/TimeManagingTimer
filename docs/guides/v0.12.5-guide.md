## 1. 시작하기

TimeKeeper는 업무 시간을 기록하고 관리하는 웹 애플리케이션입니다. 모든 데이터는 브라우저의 LocalStorage에 저장되므로 별도의 로그인이 필요 없습니다.

### 저장 용량 확인
화면 하단에서 현재 저장 용량을 확인할 수 있습니다:
```
현재 저장 용량: 123.4 KB (0.12 MB / 약 5MB 제한)
```

## 2. 주요 기능

### 타이머 상태
타이머(세션)는 4가지 상태를 가집니다:

| 상태 | 설명 | UI 표시 |
|------|------|---------|
| IDLE | 타이머 시작 전 | - |
| RUNNING | 작업 진행 중 | "진행 중" 칩 |
| PAUSED | 일시정지 상태 | 종료 시간(주황색) |
| COMPLETED | 작업 완료 | 종료 시간(기본색) |

### 업무 목록 펼치기/접기
각 업무 행을 클릭하면 세션 상세 정보를 펼치거나 접을 수 있습니다:
- **∨ 아이콘**: 세션이 접힌 상태 (클릭하면 펼쳐짐)
- **∧ 아이콘**: 세션이 펼쳐진 상태 (클릭하면 접힘)

## 3. v0.12.5 변경사항

### 펼치기/접기 토글 아이콘 추가
업무 목록에서 각 행의 맨 앞에 펼치기/접기 상태를 나타내는 아이콘이 추가되었습니다.

**변경 전**
- 행을 클릭해야 펼쳐지는지 접히는지 알 수 있음
- 현재 상태를 시각적으로 확인하기 어려움

**변경 후**
- ∨ 아이콘: 세션 상세가 접혀있음 (클릭하면 펼쳐짐)
- ∧ 아이콘: 세션 상세가 펼쳐져있음 (클릭하면 접힘)
- 현재 상태를 한눈에 확인 가능

### UI 위치
```
[∨] [✓] [▶] | 프로젝트 | 기록명 | 카테고리 | 시간 | 시작-종료 | 세션 | 버튼
 ↑
 토글 아이콘 (맨 앞)
```

## 4. FAQ

### Q: 토글 아이콘을 클릭해도 펼쳐지나요?
네, 행 전체를 클릭하면 펼쳐지고 접힙니다. 토글 아이콘은 현재 상태를 시각적으로 표시하는 역할을 합니다.

### Q: 토글 아이콘이 안 보여요
브라우저를 새로고침하면 v0.12.5 변경사항이 적용됩니다.

### Q: 여러 업무를 동시에 펼칠 수 있나요?
네, 각 업무는 독립적으로 펼치거나 접을 수 있습니다.
