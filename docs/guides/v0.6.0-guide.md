## 2. 테마 설정

앱의 색상 테마를 변경할 수 있습니다.

### 프리셋 테마

설정 > 테마 커스터마이징에서 미리 정의된 테마를 선택합니다:

| 테마 | 색상 |
|------|------|
| 기본 (검정) | 검정 |
| 초록 | 에메랄드 그린 |
| 보라 | 바이올렛 |
| 빨강 | 레드 |
| 주황 | 오렌지 |
| 청록 | 시안 |
| 파랑 | 블루 |

### 커스텀 색상

원하는 색상을 HEX 코드로 직접 입력할 수 있습니다.

1. "커스텀 색상 (HEX)" 입력란에 색상 코드 입력
2. 예: `#ff6600` (주황색)
3. 입력 즉시 미리보기 적용
4. "설정 저장" 클릭으로 영구 저장

### 테마 저장

- **저장 버튼**: 테마 설정을 LocalStorage에 저장
- **브라우저 재시작**: 저장된 테마가 자동으로 적용
- **기본값 복원**: 검정 테마로 되돌리기

## 3. 단축키 사용법

키보드로 앱을 빠르게 조작할 수 있습니다.

### 기본 단축키

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `F8` | 새 작업 추가 | 어디서든 새 작업 팝업 열기 |
| `Enter` | 타이머 시작 | 입력창에서 바로 시작 |

### Alt 조합 단축키

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Alt + N` | 입력창 포커스 | 다른 페이지에서도 일간 페이지로 이동 |
| `Alt + S` | 일시정지/재개 | 진행 중인 타이머 토글 |
| `Alt + 1` | 일간 페이지 | 일간 타이머 페이지로 이동 |
| `Alt + 2` | 주간 페이지 | 주간 일정 페이지로 이동 |
| `Alt + T` | 오늘 이동 | 일간 페이지에서 오늘로 이동 |

### 사용 예시

1. **빠른 작업 시작**:
   - `F8` → 제목 입력 → `Enter`

2. **타이머 제어**:
   - 작업 중 `Alt + S`로 일시정지
   - 다시 `Alt + S`로 재개

3. **페이지 이동**:
   - `Alt + 1`로 일간 페이지
   - `Alt + 2`로 주간 페이지

## 4. 데이터 초기화

모든 데이터를 삭제하고 앱을 초기 상태로 되돌립니다.

### 삭제되는 데이터

- 모든 작업 기록 (타이머 로그)
- 설정 (테마, 점심시간 등)
- 프리셋 (즐겨찾기, 수동 프리셋)
- 건의사항 게시글

### 초기화 방법

1. 설정 > 데이터 관리 섹션으로 이동
2. "모든 데이터 초기화" 버튼 클릭
3. 확인 모달에서 **"초기화"** 텍스트 입력
4. "초기화 실행" 클릭

### 주의사항

- **이 작업은 되돌릴 수 없습니다!**
- 초기화 전 "데이터 내보내기"로 백업을 권장합니다.
- 초기화 후 페이지가 새로고침됩니다.

## 데이터 백업 가이드

### 내보내기

1. 설정 > 데이터 관리로 이동
2. "데이터 내보내기 (JSON)" 클릭
3. `timekeeper-backup-YYYY-MM-DD.json` 파일 다운로드

### 가져오기

1. "데이터 가져오기" 클릭
2. 백업 파일 선택
3. 자동으로 데이터 복원 및 새로고침

### 백업 권장 주기

- 주 1회 정기 백업
- 브라우저 업데이트 전
- 캐시 삭제 전
